# ID3D11FunctionReflection::GetDesc


## -description


Fills the function descriptor structure for the function.


## -parameters




### -param pDesc [out]

Type: <b><a href="https://msdn.microsoft.com/1C163EE4-DA67-4341-ACA3-13ACCDD1E952">D3D11_FUNCTION_DESC</a>*</b>

A pointer to a <a href="https://msdn.microsoft.com/1C163EE4-DA67-4341-ACA3-13ACCDD1E952">D3D11_FUNCTION_DESC</a> structure that receives a description of the function.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

Returns one of the <a href="https://msdn.microsoft.com/c0856a58-b760-44e5-8acf-145720b403d1">Direct3D 11 Return Codes</a>.
