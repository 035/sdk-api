# CLUS_DNN_SODAFS_CLONE_STATUS structure


## -description


Represents the status of a Scale-Out File Server clone.


## -struct-fields




### -field NodeId

The node ID of the clone.


### -field Status

A <a href="https://msdn.microsoft.com/bd5dee18-a06f-4e46-a27e-c907b1c25a68">CLUSTER_RESOURCE_STATE</a> enumeration value that specifies the status of the clone.
