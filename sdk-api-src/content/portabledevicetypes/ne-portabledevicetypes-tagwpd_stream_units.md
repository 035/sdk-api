# tagWPD_STREAM_UNITS enumeration


## -description


The <b>WPD_STREAM_UNITS</b> enumeration specifies the unit types to be used for <a href="https://msdn.microsoft.com/4E9B2ECC-91D0-4D4F-980D-54AC88F3175D">IPortableDeviceUnitsStream</a> operations.


## -enum-fields




### -field WPD_STREAM_UNITS_BYTES

The stream units are specified in bytes.


### -field WPD_STREAM_UNITS_FRAMES

The stream units are specified in frames.


### -field WPD_STREAM_UNITS_ROWS

The stream units are specified in rows.


### -field WPD_STREAM_UNITS_MILLISECONDS

The stream units are specified in milliseconds.


### -field WPD_STREAM_UNITS_MICROSECONDS

The stream units are specified in microseconds.
