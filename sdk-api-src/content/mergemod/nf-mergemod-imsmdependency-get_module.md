# IMsmDependency::get_Module


## -description


The read-only 
<b>Module</b> property of the 
<a href="https://msdn.microsoft.com/3157f07d-99de-4628-9b03-eb86eb4896a4">Dependency Object</a> returns the ModuleID of the module that is required by the current string in the form of a <b>BSTR</b>. The ModuleID is the same form that is used in the 
<a href="https://msdn.microsoft.com/09802282-72ad-43f1-8cce-4cdc68b01e87">ModuleSignature Table</a>.

This property is read-only.
