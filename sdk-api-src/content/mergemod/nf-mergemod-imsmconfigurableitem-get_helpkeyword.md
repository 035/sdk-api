# IMsmConfigurableItem::get_HelpKeyword


## -description


The 
<b>HelpKeyword</b> property returns the value from the HelpKeyword column of the 
<a href="https://msdn.microsoft.com/3b77cc23-c104-4adc-868c-3aa2b5794bc7">ModuleConfiguration table</a>.

This property is read-only.
