# _FIXED structure


## -description



The <b>FIXED</b> structure contains the integral and fractional parts of a fixed-point real number.




## -struct-fields




### -field value

The integer part of the number.


### -field fract

The fractional part of the number.


## -remarks



The <b>FIXED</b> structure is used to describe the elements of the <a href="https://msdn.microsoft.com/841883d6-bc4d-46ef-abf4-f179771d255b">MAT2</a> structure.
