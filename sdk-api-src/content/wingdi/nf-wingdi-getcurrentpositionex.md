# GetCurrentPositionEx function


## -description


The <b>GetCurrentPositionEx</b> function retrieves the current position in logical coordinates.


## -parameters




### -param hdc [in]

A handle to the device context.


### -param lppt

TBD




#### - lpPoint [out]

A pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff569161">POINT</a> structure that receives the logical coordinates of the current position.


## -returns



If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero.
