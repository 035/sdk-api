# tagWpdParameterAttributeForm enumeration


## -description


The <a href="https://msdn.microsoft.com/7b408c9f-8b72-40aa-b691-e1630d8c2706">WpdParameterAttributeForm</a> enumeration type describes how a (method or event) parameter stores its value.


## -enum-fields




### -field WPD_PARAMETER_ATTRIBUTE_FORM_UNSPECIFIED

The form of the parameter is not specified.


### -field WPD_PARAMETER_ATTRIBUTE_FORM_RANGE

The parameter specifies a range.


### -field WPD_PARAMETER_ATTRIBUTE_FORM_ENUMERATION

The parameter is an enumeration.


### -field WPD_PARAMETER_ATTRIBUTE_FORM_REGULAR_EXPRESSION

The parameter is a regular expression.


### -field WPD_PARAMETER_ATTRIBUTE_FORM_OBJECT_IDENTIFIER




#### - WPD_PARAMETER_ATTRIBUTE_OBJECT_IDENTIFIER

The parameter is an object identifier.
