# IFaxReceiptOptions::get_AllowedReceipts


## -description


The <b>AllowedReceipts</b> property is a value that specifies the permitted types of delivery receipts.

This property is read/write.


## -parameters


## -remarks



To read or to write to this property, a user must have the <a href="https://msdn.microsoft.com/library/ms689205(v=VS.85).aspx">farQUERY_CONFIG</a> access right.
