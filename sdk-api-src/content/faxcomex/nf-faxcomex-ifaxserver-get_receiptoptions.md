# IFaxServer::get_ReceiptOptions


## -description


The <b>ReceiptOptions</b> property creates a <a href="https://msdn.microsoft.com/en-us/library/ms690118(v=VS.85).aspx">FaxReceiptOptions</a> configuration object. The object permits a fax client application to set and retrieve the receipt configuration that the fax service uses to send fax receipts.

This property is read-only.
