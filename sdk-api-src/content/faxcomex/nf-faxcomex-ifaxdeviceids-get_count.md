# IFaxDeviceIds::get_Count


## -description


The <b>Count</b> property represents the number of objects in the <a href="https://msdn.microsoft.com/library/ms686501(v=VS.85).aspx">FaxDeviceIds</a> collection. This is the total number of device IDs associated with the fax server.

This property is read-only.
