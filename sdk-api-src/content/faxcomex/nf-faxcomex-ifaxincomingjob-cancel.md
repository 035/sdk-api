# IFaxIncomingJob::Cancel


## -description


The <b>Cancel</b> method cancels the incoming fax job.


## -parameters






## -remarks



To use this method, a user must have the <a href="https://msdn.microsoft.com/70d729c6-8299-47d7-8dea-f7c754a25531">farMANAGE_JOBS</a> access right.
