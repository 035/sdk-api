# IFaxOutboundRoutingRules::get_Item


## -description


The <b>Item</b> property returns a <a href="https://msdn.microsoft.com/e027093a-314c-4292-b591-29c2bc58c031">FaxOutboundRoutingRule</a> object from the <a href="https://msdn.microsoft.com/971819e1-a3cb-4e58-b533-fee15ccb4352">FaxOutboundRoutingRules</a> collection using the routing rule's index.

This property is read-only.
