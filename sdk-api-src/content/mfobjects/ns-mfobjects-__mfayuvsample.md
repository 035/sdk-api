# __MFAYUVSample structure


## -description



Describes a 4:4:4:4 Y'Cb'Cr' sample.




## -struct-fields




### -field bCrValue

Cr (chroma difference) value.


### -field bCbValue

Cb (chroma difference) value.


### -field bYValue

Y (luma) value.


### -field bSampleAlpha8

Alpha value.
