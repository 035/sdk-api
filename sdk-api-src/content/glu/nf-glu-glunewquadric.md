# gluNewQuadric function


## -description


The <b>gluNewQuadric</b> function creates a quadric object.


## -parameters






## -remarks



The <b>gluNewQuadric</b> function creates and returns a pointer to a new quadric object. Refer to this object when calling quadric rendering and control functions. A return value of zero means there is not enough memory to allocate to the object.
