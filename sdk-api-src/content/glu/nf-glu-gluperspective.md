# gluPerspective function


## -description


The <b>gluPerspective</b> function sets up a perspective projection matrix.


## -parameters




### -param fovy

The field of view angle, in degrees, in the y-direction.


### -param aspect

The aspect ratio that determines the field of view in the x-direction. The aspect ratio is the ratio of <i>x</i> (width) to <i>y</i> (height).


### -param zNear

The distance from the viewer to the near clipping plane (always positive).


### -param zFar

The distance from the viewer to the far clipping plane (always positive).


## -returns



This function does not return a value.




## -remarks



The <b>gluPerspective</b> function specifies a viewing frustum into the world coordinate system. In general, the aspect ratio in <b>gluPerspective</b> should match the aspect ratio of the associated viewport. For example, <i>aspect</i> = 2.0 means the viewer's angle of view is twice as wide in <i>x</i> as it is in <i>y</i>. If the viewport is twice as wide as it is tall, it displays the image without distortion.

The matrix generated by <b>gluPerspective</b> is multiplied by the current matrix, just as if <a href="https://msdn.microsoft.com/0d076023-03e3-4ced-8e0a-ebff903ffdec">glMultMatrix</a> were called with the generated matrix. To load the perspective matrix onto the current matrix stack instead, precede the call to <b>gluPerspective</b> with a call to <a href="https://msdn.microsoft.com/59273aa9-4db3-4c8c-8364-f54c03d2f97a">glLoadIdentity</a>.
