# _tag_AMVABUFFERINFO structure


## -description



          The <b>AMVABUFFERINFO</b> structure specifies a buffer for the 
        <a href="https://msdn.microsoft.com/12794739-9120-4dc1-b95d-6d390d25726b">IAMVideoAccelerator::Execute</a>  method.
