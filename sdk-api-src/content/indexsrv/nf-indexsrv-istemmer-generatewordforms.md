# IStemmer::GenerateWordForms


## -description


Generates alternative forms for a word and puts these forms in the <a href="https://msdn.microsoft.com/81D52B0C-BADD-48C0-85DB-57CA82D7BBA8">IWordFormSink</a> object.


## -parameters




### -param pwcInBuf [in]

Type: <b>const WCHAR*</b>

Pointer to a buffer that contains the word to stem. The characters in <i>pwcInBuf</i> are represented in Unicode.


### -param cwc [in]

Type: <b>ULONG</b>

Number of characters in <i>pwcInBuf</i>.


### -param pStemSink






#### - pWordFormSink [in]

Type: <b>IWordFormSink*</b>

Pointer to an <a href="https://msdn.microsoft.com/81D52B0C-BADD-48C0-85DB-57CA82D7BBA8">IWordFormSink</a> object that receives and handles the alternative word forms generated by this method.
