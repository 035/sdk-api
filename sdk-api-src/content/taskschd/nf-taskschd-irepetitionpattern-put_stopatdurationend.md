# IRepetitionPattern::put_StopAtDurationEnd


## -description


Gets or sets a Boolean value that indicates if  a running instance of the task is stopped at the end of the repetition pattern duration.

This property is read/write.


## -parameters


## -remarks



When reading or writing XML for a task, this information is specified in the <a href="https://msdn.microsoft.com/4e34b5b2-ac93-4951-9de4-3e89614517d1">StopAtDurationEnd</a> element of the Task Scheduler schema.
