# ITaskSettings2::get_UseUnifiedSchedulingEngine


## -description


Gets or sets a Boolean value that indicates that the Unified Scheduling Engine will be utilized to run this task.

This property is read/write.


## -parameters


## -remarks



When reading or writing XML for a task, this setting is specified in the <a href="https://msdn.microsoft.com/93436f14-1caf-4ec8-bf74-a198b7dcb27c">UseUnifiedSchedulingEngine</a> element of the Task Scheduler schema.
