# _Application::Show


## -description


The 
<b>Show</b> method causes the MMC application to be viewable. This method sets the 
<a href="https://msdn.microsoft.com/438757ad-0ca3-4aab-b955-7c7564028f9f">Visible</a> property to 1.


## -parameters






## -returns



This method does not return a value.
