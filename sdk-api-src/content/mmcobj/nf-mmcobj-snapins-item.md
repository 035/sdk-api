# SnapIns::Item


## -description


The 
<b>Item</b> method returns the 
<a href="https://msdn.microsoft.com/94a9c623-8778-4010-bb69-6ac0d7ae5ca9">SnapIn</a> object at a specified index.


## -parameters




### -param Index

The index of the 
<a href="https://msdn.microsoft.com/94a9c623-8778-4010-bb69-6ac0d7ae5ca9">SnapIn</a> being retrieved. The index is 1-based.
