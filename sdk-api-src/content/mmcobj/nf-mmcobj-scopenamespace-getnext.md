# ScopeNamespace::GetNext


## -description


The 
<b>GetNext</b> method returns the next child sibling node for a given scope node. Before calling this method, the caller must retrieve the first child node by calling the 
<a href="https://msdn.microsoft.com/955b35fb-4dbf-4e58-ba90-4dbc93367746">ScopeNamespace.GetChild method</a>.


## -parameters




### -param Node

The scope node being queried.
