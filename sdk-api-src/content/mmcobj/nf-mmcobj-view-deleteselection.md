# View::DeleteSelection


## -description


The 
<b>DeleteSelection</b> method executes the 
<b>Delete</b> command for the selected items. If the context menu does not support 
<b>Delete</b>, then this method will not perform any operation.


## -parameters






## -returns



This method does not return a value.
