# ScopeNamespace::GetChild


## -description


The 
<b>GetChild</b> method returns the first child node for a given scope node. Use the 
<a href="https://msdn.microsoft.com/21d06208-e84d-4f98-9a69-56ba06004c82">ScopeNamespace.GetNext method</a> to retrieve the siblings of the node returned by the 
<b>GetChild</b> method.


## -parameters




### -param Node

The scope node being queried.


### -param Child






## -remarks



For an example of code that uses the 
<b>GetChild</b> method, see the 
<a href="https://msdn.microsoft.com/21d06208-e84d-4f98-9a69-56ba06004c82">ScopeNamespace.GetNext</a> method.
