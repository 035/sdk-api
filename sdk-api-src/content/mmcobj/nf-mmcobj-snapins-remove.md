# SnapIns::Remove


## -description


The 
<b>Remove</b> method removes a 
<a href="https://msdn.microsoft.com/94a9c623-8778-4010-bb69-6ac0d7ae5ca9">SnapIn</a> object from the collection. Removing the snap-in from the collection removes it from the console as well.


## -parameters




### -param SnapIn

The 
<a href="https://msdn.microsoft.com/94a9c623-8778-4010-bb69-6ac0d7ae5ca9">SnapIn</a> object that represents the snap-in to remove from the collection.


## -returns



This method does not return a value.
