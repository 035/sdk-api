# View::Select


## -description


The 
<b>Select</b> method selects the specified node in the result pane.


## -parameters




### -param Node

The node, as a <a href="https://msdn.microsoft.com/c504cb3f-8ec6-49fe-b0d3-05ea9cad61f4">Node</a> object, to be selected.


## -returns



This method does not return a value.
