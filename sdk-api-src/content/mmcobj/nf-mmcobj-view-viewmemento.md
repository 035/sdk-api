# View::ViewMemento


## -description


The 
<b>ViewMemento</b> method sets the view to the specified memento's state. A memento is the programmatic equivalent of a favorite setting.


## -parameters




### -param Memento

The XML-readable string that represents the view's state.


## -returns



This method does not return a value.
