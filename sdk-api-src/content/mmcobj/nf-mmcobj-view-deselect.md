# View::Deselect


## -description


The 
<b>Deselect</b> method removes the specified node from the current selection.


## -parameters




### -param Node

A value that specifies the node, as a <a href="https://msdn.microsoft.com/c504cb3f-8ec6-49fe-b0d3-05ea9cad61f4">Node</a> object,  to be removed from the current selection.


## -returns



This method does not return a value.
