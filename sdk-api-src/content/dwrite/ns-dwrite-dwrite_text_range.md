# DWRITE_TEXT_RANGE structure


## -description


Specifies a range of text positions where format is applied in the text represented by an <a href="https://msdn.microsoft.com/0d687337-8623-4014-967c-f533072e31cc">IDWriteTextLayout</a> object.


## -struct-fields




### -field startPosition

Type: <b>UINT32</b>

The start position of the text range.


### -field length

Type: <b>UINT32</b>

The number positions in the text range.
