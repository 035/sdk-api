# IDirectManipulationViewport::RemoveEventHandler


## -description


Removes an existing event handler from the viewport.


## -parameters




### -param cookie [in]

A value that was returned by a previous call to <a href="https://msdn.microsoft.com/56b47fec-dfa2-4906-9135-5ee331f04c54">AddEventHandler</a>.


## -returns



If the method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
