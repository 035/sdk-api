# IMSVidCtl::get_State


## -description


The <b>get_State</b> method retrieves the state of the filter graph.


## -parameters




### -param lState






#### - plState [out]

Receives a member of the <a href="https://msdn.microsoft.com/b4da9c6e-3235-4c78-b9e1-57c9d06fccbc">MSVidCtlStateList</a> enumeration.
