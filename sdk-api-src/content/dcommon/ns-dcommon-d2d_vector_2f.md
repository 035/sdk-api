# D2D_VECTOR_2F structure


## -description


A 2D vector that consists of two single-precision floating-point values (x, y).




## -struct-fields




### -field x

Type: <b>FLOAT</b>

The x value of the vector.


### -field y

Type: <b>FLOAT</b>

The y value of the vector.
