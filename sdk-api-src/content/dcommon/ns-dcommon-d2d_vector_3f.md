# D2D_VECTOR_3F structure


## -description


A 3D vector that consists of three single-precision floating-point values (x, y, z).


## -struct-fields




### -field x

Type: <b>FLOAT</b>

The x value of the vector.


### -field y

Type: <b>FLOAT</b>

The y value of the vector.


### -field z

Type: <b>FLOAT</b>

The z value of the vector.
