# DXGI_OVERLAY_SUPPORT_FLAG enumeration


## -description


Specifies overlay support to check for in a call to <a href="https://msdn.microsoft.com/D7B90FF5-5E8B-4F9E-A442-B44449438388">IDXGIOutput3::CheckOverlaySupport</a>.


## -enum-fields




### -field DXGI_OVERLAY_SUPPORT_FLAG_DIRECT

Direct overlay support.


### -field DXGI_OVERLAY_SUPPORT_FLAG_SCALING

Scaling overlay support.
