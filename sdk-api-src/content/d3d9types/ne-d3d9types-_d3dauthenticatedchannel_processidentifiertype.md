# _D3DAUTHENTICATEDCHANNEL_PROCESSIDENTIFIERTYPE enumeration


## -description


Specifies the type of process that is identified in the <a href="https://msdn.microsoft.com/763c56b5-b240-4bad-b601-07959ed37479">D3DAUTHENTICATEDCHANNEL_QUERYRESTRICTEDSHAREDRESOURCEPROCESS_OUTPUT</a> structure.


## -enum-fields




### -field PROCESSIDTYPE_UNKNOWN

Unknown process type.


### -field PROCESSIDTYPE_DWM

Desktop Window Manager (DWM) process.


### -field PROCESSIDTYPE_HANDLE

Handle to a process.
