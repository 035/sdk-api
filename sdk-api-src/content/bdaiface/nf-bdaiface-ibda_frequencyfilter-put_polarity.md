# IBDA_FrequencyFilter::put_Polarity


## -description



The <b>put_Polarity</b> method specifies the polarity of the signal.




## -parameters




### -param Polarity [in]

Specifies the polarity as a member of the <a href="https://msdn.microsoft.com/19303d43-913f-4719-b1f8-e0e785ac0483">Polarisation</a> enumerated type.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
