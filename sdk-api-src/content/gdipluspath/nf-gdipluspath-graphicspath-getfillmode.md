# GraphicsPath::GetFillMode


## -description


The <b>GraphicsPath::GetFillMode</b> method gets the fill mode of this path.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/fd54e07b-74a2-4bce-9fa3-e36afec4be92">FillMode</a></b>
</strong>

This method returns an element of the <a href="https://msdn.microsoft.com/fd54e07b-74a2-4bce-9fa3-e36afec4be92">FillMode</a> enumeration.
