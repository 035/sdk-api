# GraphicsPath::GetFillMode


## -description


The <b>GraphicsPath::GetFillMode</b> method gets the fill mode of this path.

## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/library/ms534120(v=VS.85).aspx">FillMode</a></b>
</strong>
