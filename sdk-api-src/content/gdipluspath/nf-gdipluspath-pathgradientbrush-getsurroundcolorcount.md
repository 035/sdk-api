# PathGradientBrush::GetSurroundColorCount


## -description


The <b>PathGradientBrush::GetSurroundColorCount</b> method gets the number of colors that have been specified for the boundary path of this path gradient brush.


## -parameters






## -returns



Type: <strong>Type: <b>INT</b>
</strong>

This method returns the number of colors that have been specified for the boundary path of this path gradient brush.
