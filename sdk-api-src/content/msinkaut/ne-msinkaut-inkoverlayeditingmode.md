# InkOverlayEditingMode enumeration


## -description



Specifies the behavior mode of the <a href="https://msdn.microsoft.com/61191ab3-075e-458b-9e0f-4bc255687b3c">InkOverlay</a> object and the <a href="https://msdn.microsoft.com/e9fa6807-6e2a-44ec-9b8f-a560185e4367">InkPicture</a> control.




## -enum-fields




### -field IOEM_Ink

The object or control is in collection mode.


### -field IOEM_Delete

The object or control is in deletion mode.


### -field IOEM_Select

The object or control is in selection and editing mode.
