# IsMenu function


## -description


Determines whether a handle is a menu handle. 


## -parameters




### -param hMenu [in]

Type: <b>HMENU</b>

A handle to be tested. 


## -returns



Type: <b>BOOL</b>

If the handle is a menu handle, the return value is nonzero. 

If the handle is not a menu handle, the return value is zero.
