# IVMRAspectRatioControl9::SetAspectRatioMode


## -description



The <code>SetAspectRatioMode</code> method specifies whether the VMR preserves the aspect ratio of the source video.




## -parameters




### -param dwARMode






#### - AspectRatioMode [in]

Specifies a member of the <a href="https://msdn.microsoft.com/dd1d1d99-008b-4234-a38a-314ba02bb116">VMR_ASPECT_RATIO_MODE</a> enumeration type.
