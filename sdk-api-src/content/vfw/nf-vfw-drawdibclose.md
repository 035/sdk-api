# DrawDibClose function


## -description



The <b>DrawDibClose</b> function closes a DrawDib DC and frees the resources DrawDib allocated for it.




## -parameters




### -param hdd

Handle to a DrawDib DC.


## -returns



Returns <b>TRUE</b> if successful or <b>FALSE</b> otherwise.
