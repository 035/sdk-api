# DXGI_HDR_METADATA_TYPE enumeration


## -description


Specifies the header metadata type.


## -enum-fields




### -field DXGI_HDR_METADATA_TYPE_NONE

Indicates there is no header metadata.


### -field DXGI_HDR_METADATA_TYPE_HDR10

Indicates the header metadata is held by a  <a href="https://msdn.microsoft.com/67A53A43-121F-4D83-AACC-D25D58123BE1">DXGI_HDR_METADATA_HDR10</a> structure.


### -field DXGI_HDR_METADATA_TYPE_HDR10PLUS




## -remarks



This enum is used by the <a href="https://msdn.microsoft.com/03EBBB29-EAC3-4FE7-9CA7-D9F62CFDA8FB">SetHDRMetaData</a> method.
