# _DXGI_OFFER_RESOURCE_FLAGS enumeration


## -description


Specifies flags for the <a href="https://msdn.microsoft.com/7F6782F3-7779-4DBD-AD5A-AE0FB136FC70">OfferResources1</a> method.


## -enum-fields




### -field DXGI_OFFER_RESOURCE_FLAG_ALLOW_DECOMMIT

Indicates the ability to allow memory de-commit by the DirectX Graphics Kernel.
