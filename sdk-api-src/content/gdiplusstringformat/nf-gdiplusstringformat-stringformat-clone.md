# StringFormat::Clone


## -description


The <b>StringFormat::Clone</b> method creates a new 
			<a href="https://msdn.microsoft.com/2d7af5fe-f3e9-4db3-90a5-4e623d9ce773">StringFormat</a> object and initializes it with the contents of this 
			<b>StringFormat</b> object.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/2d7af5fe-f3e9-4db3-90a5-4e623d9ce773">StringFormat</a>*</b>
</strong>

This method returns a pointer to the new 
						<a href="https://msdn.microsoft.com/2d7af5fe-f3e9-4db3-90a5-4e623d9ce773">StringFormat</a> object.
