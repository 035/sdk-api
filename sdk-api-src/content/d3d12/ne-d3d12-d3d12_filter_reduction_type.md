# D3D12_FILTER_REDUCTION_TYPE enumeration


## -description


Specifies the type of filter reduction.




## -enum-fields




### -field D3D12_FILTER_REDUCTION_TYPE_STANDARD

The filter type is standard.


### -field D3D12_FILTER_REDUCTION_TYPE_COMPARISON

The filter type is comparison.


### -field D3D12_FILTER_REDUCTION_TYPE_MINIMUM

The filter type is minimum.


### -field D3D12_FILTER_REDUCTION_TYPE_MAXIMUM

The filter type is maximum.


## -remarks



This enum is used by the <a href="https://msdn.microsoft.com/96261FE1-89D4-4135-B5C4-2D788DF4FA12">D3D12_SAMPLER_DESC</a> structure. Also, refer to the remarks for <a href="https://msdn.microsoft.com/3755A722-34E5-415E-8760-93094D033E05">D3D12_FILTER</a>.
