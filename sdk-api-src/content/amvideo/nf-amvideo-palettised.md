# PALETTISED macro


## -description


The <code>PALETTISED</code> macro checks whether a bitmap has a color depth of 8 bits or less. 


## -parameters




### -param pbmi

Pointer to a <a href="https://msdn.microsoft.com/a175592b-0dc1-4001-b52f-785407965932">VIDEOINFOHEADER</a> structure.
