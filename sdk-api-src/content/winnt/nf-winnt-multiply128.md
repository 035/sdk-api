# Multiply128 function


## -description


Multiplies two 64-bit integers to produce a 128-bit integer.


## -parameters




### -param Multiplier [in]

The first integer.


### -param Multiplicand [in]

The second integer.


### -param HighProduct [out]

The high 64 bits of the product.


## -returns



The low 64 bits of the product.
