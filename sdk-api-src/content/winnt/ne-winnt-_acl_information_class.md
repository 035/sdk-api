# _ACL_INFORMATION_CLASS enumeration


## -description


The <b>ACL_INFORMATION_CLASS</b> enumeration contains values that specify the type of information being assigned to or retrieved from an <a href="https://msdn.microsoft.com/0baaa937-f635-4500-8dcd-9dbbd6f4cd02">access control list</a> (ACL).


## -enum-fields




### -field AclRevisionInformation

Indicates ACL revision information.


### -field AclSizeInformation

Indicates ACL size information.
