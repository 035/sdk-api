# _ACL_REVISION_INFORMATION structure


## -description


The <b>ACL_REVISION_INFORMATION</b> structure contains revision information about an <a href="https://msdn.microsoft.com/library/windows/hardware/ff538866">ACL</a> structure.


## -struct-fields




### -field AclRevision

Specifies a revision number. The current revision number is ACL_REVISION.
