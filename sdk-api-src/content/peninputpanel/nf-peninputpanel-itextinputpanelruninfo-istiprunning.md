# ITextInputPanelRunInfo::IsTipRunning


## -description



Indicates if the Tablet PC Input Panel is running at the time the method is called.




## -parameters




### -param pfRunning [out]

<b>TRUE</b> if the Input Panel was running, otherwise <b>FALSE</b>.
