# D2D1_SVG_VIEWBOX structure


## -description


Represents an SVG viewBox.


## -struct-fields




### -field x

X coordinate of the view box.


### -field y

Y coordinate of the view box.


### -field width

Width of the view box.


### -field height

Height of the view box.
