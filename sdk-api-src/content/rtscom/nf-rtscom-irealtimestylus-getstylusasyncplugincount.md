# IRealTimeStylus::GetStylusAsyncPluginCount


## -description



Retrieves the number of plug-ins in the asynchronous plug-in collection.




## -parameters




### -param pcPlugins






#### - pcPlugin [out]

 The plug-in count for the asynchronous plug-in collection.


## -returns



For a description of the return values, see <a href="https://msdn.microsoft.com/fc0900b4-f08b-4a93-bbc0-d3db067d7917">RealTimeStylus Classes and Interfaces</a>.
