# StylusInfo structure


## -description



Provides information about the stylus.




## -struct-fields




### -field tcid

Uniquely identifies the tablet.


### -field cid

Uniquely identifies the stylus.


### -field bIsInvertedCursor

<b>TRUE</b> if the stylus is upside down, otherwise <b>FALSE</b>.
