# ICertificateEnrollmentServerSetup::get_ErrorString


## -description


The <b>ErrorString</b> property retrieves a string that contains additional information about Certificate Enrollment Web Service (CES) setup failure.

This property is read-only.


## -parameters


## -remarks



Calling any method on the <a href="https://msdn.microsoft.com/B25DA7C4-0503-4E3B-BABC-6EFBD9EBDDAE">ICertificateEnrollmentServerSetup</a> interface resets this property value to an empty error string.
