# tagSTYLEBUFW structure


## -description



Contains the identifier and name of a style.




## -struct-fields




### -field dwStyle

Style of the register string. Can be IME_REGWORD_STYLE_EUDC to indicate a string in the EUDC range.


### -field szDescription

Description of the style.
