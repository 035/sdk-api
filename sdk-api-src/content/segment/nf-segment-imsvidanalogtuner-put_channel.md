# IMSVidAnalogTuner::put_Channel


## -description


The <b>put_Channel</b> method specifies the tuner's channel.


## -parameters




### -param Channel [in]

Specifies the channel.


## -returns



If the method succeeds, it returns S_OK. If it fails, it returns an error code.
