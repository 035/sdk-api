# CMSPCallMultiGraph function


## -description


The 
<b>FinalRelease</b> method <a href="https://msdn.microsoft.com/317e3fba-3af5-4fd6-8e4d-276643960930">CMSPCallMultiGraph</a>releases the call object's reference to the address object. This is postponed to 
<b>FinalRelease</b> in case the stream objects hold references on the call object during the shutdown process.
