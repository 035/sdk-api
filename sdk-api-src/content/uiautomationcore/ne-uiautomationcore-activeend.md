# ActiveEnd enumeration


## -description


Contains possible values for the <a href="uiauto_textattribute_ids.htm">SelectionActiveEnd</a> text attribute, which indicates the location of the caret relative to a text range that represents the currently selected text.


## -enum-fields




### -field ActiveEnd_None

The caret is not at either end of the text range.


### -field ActiveEnd_Start

The caret is at the beginning of the text range.


### -field ActiveEnd_End

The caret is at the end of the text range.
