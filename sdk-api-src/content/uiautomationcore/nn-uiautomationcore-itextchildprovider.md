# ITextChildProvider interface


## -description


Provides access to a text-based control (or an object embedded in text) that is a child or descendant of another text-based control. 


## -remarks




Implemented on a descendant of an element that supports the <a href="https://msdn.microsoft.com/4d541c31-11f7-4d7e-a0e0-9ed1da873d07">Text</a> control pattern. That is, an element that implements <a href="https://msdn.microsoft.com/B33BCBEF-9AD2-4A5A-871E-E97E69BE8195">TextChild</a> typically also supports the Text control pattern.
