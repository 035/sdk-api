# _WS_XML_WRITER_STREAM_OUTPUT structure


## -description



        Specifies that the generated bytes should be sent to callback.
      


## -struct-fields




### -field output


          The base type for all types that derive from <a href="https://msdn.microsoft.com/367e6f98-9351-4a08-b8ce-036e7f2788e4">WS_XML_WRITER_OUTPUT</a>.
        


### -field writeCallback


          A callback that will get invoked when <a href="https://msdn.microsoft.com/ba631942-d5a0-4d93-9899-c3f0ebd4aae5">WsFlushWriter</a> is called.  This may not be <b>NULL</b>.
        


### -field writeCallbackState


          User-defined data that will be passed to the writeCallback.
