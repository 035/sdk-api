# ITfUIElementSink::EndUIElement


## -description


The <b>ITfUIElementSink::EndUIElement</b> method is called when the UIElement is finished.


## -parameters




### -param dwUIElementId [in]

[in] Id of the UIElement that is finished.


## -returns



The TSF manager ignores the return value of this method.
