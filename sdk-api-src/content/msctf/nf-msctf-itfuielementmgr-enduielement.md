# ITfUIElementMgr::EndUIElement


## -description


The <b>ITfUIElementMgr::EndUIElement</b> method is called by a text service when the element of UI is hidden.


## -parameters




### -param dwUIElementId [in]

[in] The element id to hide the UI element.
