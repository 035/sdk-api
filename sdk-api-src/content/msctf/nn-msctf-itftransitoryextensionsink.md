# ITfTransitoryExtensionSink interface


## -description


The <b>ITfTransitoryExtensionSink</b> interface is implemented by the application that uses Transitory Extension dim. The application can track the changes that happened in the transitory extension by using this sink interface.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ITfTransitoryExtensionSink</b> interface inherits from the <a href="https://msdn.microsoft.com/33f1d79a-33fc-4ce5-a372-e08bda378332">IUnknown</a> interface. <b>ITfTransitoryExtensionSink</b> also has these types of members:
