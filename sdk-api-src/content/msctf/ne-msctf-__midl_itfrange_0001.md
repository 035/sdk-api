# __MIDL_ITfRange_0001 enumeration


## -description


Elements of the <b>TfGravity</b> enumeration specify the type of gravity associated with the anchor of an <a href="https://msdn.microsoft.com/b8889f7d-3228-4ecc-8d24-c04234d3101e">ITfRange</a> object.


## -enum-fields




### -field TF_GRAVITY_BACKWARD

The anchor has backward gravity.


### -field TF_GRAVITY_FORWARD

The anchor has forward gravity.


## -remarks



For more information about anchor gravity, see <a href="ranges.htm">Anchor Gravity</a>.
