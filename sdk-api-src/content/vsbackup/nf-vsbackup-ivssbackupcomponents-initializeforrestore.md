# IVssBackupComponents::InitializeForRestore


## -description


The 
<b>InitializeForRestore</b> method initializes the 
<a href="https://msdn.microsoft.com/fe1220c7-11e5-4872-b7a9-61558f7c75c0">IVssBackupComponents</a> interface in preparation for a restore operation.


## -parameters




### -param bstrXML [in]

XML string containing the Backup Components Document generated by a backup operation and saved by <a href="https://msdn.microsoft.com/8184d15a-7d1f-49ed-afe3-fa9d81a4d32d">IVssBackupComponents::SaveAsXML</a>.

## -remarks



The XML document supplied to this method initializes the 
<a href="https://msdn.microsoft.com/fe1220c7-11e5-4872-b7a9-61558f7c75c0">IVssBackupComponents</a> object with metadata previously stored by a call to 
<a href="https://msdn.microsoft.com/8184d15a-7d1f-49ed-afe3-fa9d81a4d32d">IVssBackupComponents::SaveAsXML</a>. Users should not tamper with this metadata document.
