# CXAPOBase class


## -description


Default implementation of the <a href="https://msdn.microsoft.com/library/Ee415893(v=VS.85).aspx">IXAPO</a> interface.

For a list of all members of this class, see <a href="https://msdn.microsoft.com/library/Ee415237(v=VS.85).aspx">CXAPOBase Members</a>.




## -remarks



<b>CXAPOBase</b> provides overridable implementations for all <a href="https://msdn.microsoft.com/library/Ee415893(v=VS.85).aspx">IXAPO</a> methods except <a href="https://msdn.microsoft.com/library/Ee418456(v=VS.85).aspx">IXAPO::Process</a>.

<h3><a id="Platform_Requirements"></a><a id="platform_requirements"></a><a id="PLATFORM_REQUIREMENTS"></a>Platform Requirements</h3>
Windows 10 (XAudio2.9); Windows 8, Windows Phone 8 (XAudio 2.8); DirectX SDK (XAudio 2.7)
