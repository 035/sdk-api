# DXGI_MODE_ROTATION enumeration


## -description


Flags that indicate how the back buffers should be rotated to fit the physical rotation of a monitor.


## -enum-fields




### -field DXGI_MODE_ROTATION_UNSPECIFIED

Unspecified rotation.


### -field DXGI_MODE_ROTATION_IDENTITY

Specifies no rotation.


### -field DXGI_MODE_ROTATION_ROTATE90

Specifies 90 degrees of rotation.


### -field DXGI_MODE_ROTATION_ROTATE180

Specifies 180 degrees of rotation.


### -field DXGI_MODE_ROTATION_ROTATE270

Specifies 270 degrees of rotation.
