# DXGI_JPEG_AC_HUFFMAN_TABLE structure


## -description


Describes a JPEG AC huffman table.


## -struct-fields




### -field CodeCounts

The number of codes for each code length.


### -field CodeValues

The Huffman code values, in order of increasing code length.
