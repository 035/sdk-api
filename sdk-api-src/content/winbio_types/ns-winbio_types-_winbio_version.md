# _WINBIO_VERSION structure


## -description


The <b>WINBIO_VERSION</b> structure contains the software version number  of a biometric service provider component.


## -struct-fields




### -field MajorVersion

A <b>DWORD</b> that contains the major version number.


### -field MinorVersion

A <b>DWORD</b> that contains the minor version number.
