# _WSDXML_TYPE structure


## -description


Describes an XSD type. This structure is populated by <a href="https://msdn.microsoft.com/76dffca8-bb84-4384-a9e8-120a4cf2acac">generated code</a>.


## -struct-fields




### -field Uri

The optional URI that identifies the type.


### -field Table

The type table that describes the schema of the type and its binary representation.
