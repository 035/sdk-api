# D2D1_HIGHLIGHTSANDSHADOWS_INPUT_GAMMA enumeration


## -description


Values for the <a href="https://msdn.microsoft.com/2E4BACCB-EF29-44FB-8427-C10211BC4899">D2D1_HIGHLIGHTSANDSHADOWS_PROP_INPUT_GAMMA</a> property 
        of the <a href="https://msdn.microsoft.com/ebbb7d99-9144-ffff-af73-d89e7d269924">Highlights and Shadows effect</a>.


## -enum-fields




### -field D2D1_HIGHLIGHTSANDSHADOWS_INPUT_GAMMA_LINEAR

Indicates the input image is in linear gamma space.


### -field D2D1_HIGHLIGHTSANDSHADOWS_INPUT_GAMMA_SRGB

Indicates the input image is sRGB gamma space.
