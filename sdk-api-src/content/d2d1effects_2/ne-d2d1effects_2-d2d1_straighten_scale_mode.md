# D2D1_STRAIGHTEN_SCALE_MODE enumeration


## -description


Values for the <a href="https://msdn.microsoft.com/AD22B649-E4FE-41BD-85FE-B000CCB7F48D">D2D1_STRAIGHTEN_PROP_SCALE_MODE</a> property of the <a href="https://msdn.microsoft.com/aa37cdf1-bbb6-db4e-45a7-67c7cc16b7b4">Straighten effect</a>.


## -enum-fields




### -field D2D1_STRAIGHTEN_SCALE_MODE_NEAREST_NEIGHBOR

Indicates nearest neighbor interpolation should be used.


### -field D2D1_STRAIGHTEN_SCALE_MODE_LINEAR

Indicates linear interpolation should be used.


### -field D2D1_STRAIGHTEN_SCALE_MODE_CUBIC

Indicates cubic interpolation should be used.


### -field D2D1_STRAIGHTEN_SCALE_MODE_MULTI_SAMPLE_LINEAR

Indicates multi-sample linear interpolation should be used.


### -field D2D1_STRAIGHTEN_SCALE_MODE_ANISOTROPIC

Indicates anisotropic filtering should be used.
