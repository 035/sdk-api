# D2D1_EDGEDETECTION_MODE enumeration


## -description


Values for the <a href="https://msdn.microsoft.com/F3B94BDC-120E-4B44-9AA8-595AAEFD1F61">D2D1_EDGEDETECTION_PROP_MODE</a> property of the <a href="https://msdn.microsoft.com/d22868cf-95fe-690e-66ac-268d7e116aee">Edge Detection effect</a>.


## -enum-fields




### -field D2D1_EDGEDETECTION_MODE_SOBEL

Indicates the Sobel operator should be used for edge detection.


### -field D2D1_EDGEDETECTION_MODE_PREWITT

Indicates the Prewitt operator should be used for edge detection.
