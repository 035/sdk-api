# SpatialAudioHrtfDirectivityType enumeration


## -description


Specifies the shape in which sound is emitted by an <a href="https://msdn.microsoft.com/E69F1D09-B937-4BCC-A040-18EF8A838289">ISpatialAudioObjectForHrtf</a>.


## -enum-fields




### -field SpatialAudioHrtfDirectivity_OmniDirectional

The sound is emitted in all directions.


### -field SpatialAudioHrtfDirectivity_Cardioid

The sound is emitted in a cardioid shape.


### -field SpatialAudioHrtfDirectivity_Cone

The sound is emitted in a cone shape.
