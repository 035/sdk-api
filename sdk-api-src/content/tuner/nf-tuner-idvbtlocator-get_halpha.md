# IDVBTLocator::get_HAlpha method


## -description



The <b>get_HAlpha</b> method retrieves the hierarchy alpha.




## -parameters




### -param Alpha






#### - pAlpha [out]

Receives a member of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff559667">HierarchyAlpha</a> enumeration.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
