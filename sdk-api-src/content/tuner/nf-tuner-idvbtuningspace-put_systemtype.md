# IDVBTuningSpace::put_SystemType method


## -description



The <b>put_SystemType</b> method sets the system type.




## -parameters




### -param SysType [in]

Variable of type <a href="https://msdn.microsoft.com/library/windows/hardware/ff558733">DVBSystemType</a> that specifies the system type.


## -returns



Returns S_OK if successful. If the method fails, error information can be retrieved using the standard COM <b>IErrorInfo</b> interface.
