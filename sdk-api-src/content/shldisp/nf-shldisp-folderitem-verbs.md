# FolderItem::Verbs


## -description


Retrieves the item's <a href="https://msdn.microsoft.com/31badb4b-b89e-4294-9dd7-bda716e163b2">FolderItemVerbs</a> object. This object is the collection of verbs that can be executed on the item.

### -param ppfic






## -returns



Type: <b><a href="https://msdn.microsoft.com/31badb4b-b89e-4294-9dd7-bda716e163b2">FolderItemVerbs</a>**</b>

An object reference to the <a href="https://msdn.microsoft.com/31badb4b-b89e-4294-9dd7-bda716e163b2">FolderItemVerbs</a> object.
