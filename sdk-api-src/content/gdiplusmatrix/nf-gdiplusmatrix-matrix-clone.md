# Matrix::Clone


## -description


The <b>Matrix::Clone</b> method creates a new 
			<a href="https://msdn.microsoft.com/92b0d9db-3d4c-47b8-87cd-60d7b4323f0a">Matrix</a> object that is a copy of this 
			<b>Matrix</b> object.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/92b0d9db-3d4c-47b8-87cd-60d7b4323f0a">Matrix</a>*</b>
</strong>

This method returns a pointer to the new 
						<a href="https://msdn.microsoft.com/92b0d9db-3d4c-47b8-87cd-60d7b4323f0a">Matrix</a> object.
