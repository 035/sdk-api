# Matrix::IsInvertible


## -description


The <b>Matrix::IsInvertible</b> method determines whether this matrix is invertible.


## -parameters






## -returns



Type: <strong>Type: <b>BOOL</b>
</strong>

If this matrix is invertible, this method returns <b>TRUE</b>; otherwise, it returns <b>FALSE</b>.
