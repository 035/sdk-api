# AM_PROPERTY_DVD_RATE_CHANGE enumeration


## -description



The <b>AM_PROPERTY_DVD_RATE_CHANGE</b> enumeration is not used.




## -enum-fields




### -field AM_RATE_ChangeRate

Reserved.


### -field AM_RATE_FullDataRateMax

Reserved.


### -field AM_RATE_ReverseDecode

Reserved.


### -field AM_RATE_DecoderPosition

Reserved.


### -field AM_RATE_DecoderVersion

Reserved.


## -remarks



This enumeration is defined in dvdmedia.h but was never used. It is included here for the sake of completeness. For DVD rate-change control, use the <a href="https://msdn.microsoft.com/f88c64ce-af76-49fe-8ebd-029928506243">Rate Change Property Set</a>.
