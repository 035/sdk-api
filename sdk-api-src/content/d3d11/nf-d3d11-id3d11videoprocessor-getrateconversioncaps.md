# ID3D11VideoProcessor::GetRateConversionCaps


## -description


Gets the rate conversion capabilities of the video processor.


## -parameters




### -param pCaps [out]

A pointer to a <a href="https://msdn.microsoft.com/C8C50AE4-5F4F-42AB-8FBB-37D24C4D6081">D3D11_VIDEO_PROCESSOR_RATE_CONVERSION_CAPS</a> structure that receives the rate conversion capabilities.


## -returns



This method does not return a value.
