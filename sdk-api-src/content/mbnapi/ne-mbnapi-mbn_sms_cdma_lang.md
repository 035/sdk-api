# MBN_SMS_CDMA_LANG enumeration


## -description


The <b>MBN_SMS_CDMA_LANG</b> enumerated type represents the different languages that can be used in a CDMA message.


## -enum-fields




### -field MBN_SMS_CDMA_LANG_NONE

None.


### -field MBN_SMS_CDMA_LANG_ENGLISH

English.


### -field MBN_SMS_CDMA_LANG_FRENCH

French.


### -field MBN_SMS_CDMA_LANG_SPANISH

Spanish.


### -field MBN_SMS_CDMA_LANG_JAPANESE

Japanese.


### -field MBN_SMS_CDMA_LANG_KOREAN

Korean.


### -field MBN_SMS_CDMA_LANG_CHINESE

Chinese.


### -field MBN_SMS_CDMA_LANG_HEBREW

Hebrew.
