# IFolderActionCollection::AddRange


## -description


Adds one or more folder actions to the collection.


## -parameters




### -param Actions [in]

An <a href="https://msdn.microsoft.com/9b0ab26f-7e91-4d7a-9fd7-73332601dd7b">IFolderActionCollection</a> interface to a collection of one or more folder actions to add to this collection.


## -returns



Returns S_OK if successful.
