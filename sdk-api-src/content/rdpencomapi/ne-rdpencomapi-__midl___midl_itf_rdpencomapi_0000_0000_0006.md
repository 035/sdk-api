# __MIDL___MIDL_itf_rdpencomapi_0000_0000_0006 enumeration


## -description


Defines values for the type of attendee. Attendee flags can be queried for each attendee by using the attendee list.


## -enum-fields




### -field ATTENDEE_FLAGS_LOCAL

Indicates that the attendee is a local attendee.
