# timecaps_tag structure


## -description



          The <b>TIMECAPS</b> structure contains information about the resolution of the timer.


## -struct-fields




### -field wPeriodMin


            The minimum supported resolution, in milliseconds.


### -field wPeriodMax


            The maximum supported resolution, in milliseconds.
