# __MIDL___MIDL_itf_mfidl_0000_0109_0002 enumeration


## -description


Specifies whether a stream associated with an <a href="https://msdn.microsoft.com/061EF002-178E-42CA-9D32-7E1282297BA4">IMFSensorDevice</a> is an input or an output stream.


## -enum-fields




### -field MFSensorStreamType_Unknown

The sensor stream type is unknown.


### -field MFSensorStreamType_Input

The sensor stream is an input stream.


### -field MFSensorStreamType_Output

The sensor stream is an output stream.
