# ISpatialAudioMetadataItems::GetMaxValueBufferLength method


## -description


The size of the largest command value defined by the metadata format for the <a href="https://msdn.microsoft.com/54A6B7DE-A41E-4214-AF02-CC19250B9037">ISpatialAudioMetadataItems</a>.


## -parameters




### -param maxValueBufferLength






#### - frameCount [out]

The size of the largest command value defined by the metadata format.


## -returns



If the method succeeds, it returns S_OK.
