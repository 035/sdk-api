# Session_IPv4 structure


## -description


The 
<b>Session_IPv4</b> structure stores information about an IPv4 RSVP SESSION object.


## -struct-fields




### -field sess_destaddr

Destination IP address for the session.


### -field sess_protid

Protocol ID for the session.


### -field sess_flags

Session flags. 


### -field sess_destport

Destination IP address for the session.
