# Rsvp_Hop_IPv4 structure


## -description


The 
<b>Rsvp_Hop_IPv4</b> structure stores information about an RSVP-enabled IPv4 hop.


## -struct-fields




### -field hop_ipaddr

The next or previous hop IP address, in the form of an <a href="https://msdn.microsoft.com/library/windows/hardware/ff556972">in_addr</a> structure.


### -field hop_LIH

Logical Interface Handle.
