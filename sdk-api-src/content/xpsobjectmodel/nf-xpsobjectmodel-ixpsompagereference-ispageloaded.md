# IXpsOMPageReference::IsPageLoaded


## -description


Gets the referenced page status,  which indicates whether the  page is loaded.


## -parameters




### -param isPageLoaded [out, retval]

A Boolean value that indicates the status of the page.

</dl>
</td>

The page is loaded.

</td>
</tr>
<tr>

<dt><b><b>FALSE</b></b></dt>
</dl>

</td>
</tr>
</table>
