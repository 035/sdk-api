# _DXVA2_ValueRange structure


## -description



Defines the range of supported values for a DirectX Video Acceleration (DXVA) operation.




## -struct-fields




### -field MinValue

Minimum supported value.


### -field MaxValue

Maximum supported value.


### -field DefaultValue

Default value.


### -field StepSize

Minimum increment between values.


## -remarks



All values in this structure are specified as <a href="https://msdn.microsoft.com/5f8f4515-1cf4-4060-813b-c746649c5c40">DXVA2_Fixed32</a> structures.
