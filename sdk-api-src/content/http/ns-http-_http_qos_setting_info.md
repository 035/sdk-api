# _HTTP_QOS_SETTING_INFO structure


## -description


The <b>HTTP_QOS_SETTING_INFO</b> structurecontains information about a QOS setting.


## -struct-fields




### -field QosType

An <a href="https://msdn.microsoft.com/d1593670-9f5c-48a9-93d9-d5ff744cfc8b">HTTP_QOS_SETTING_TYPE</a> enumeration value that specifies the type of the QOS setting.


### -field QosSetting

A pointer to a structure that contains the setting.
