# TextureBrush class


## -description


The <b>TextureBrush</b> class defines a <a href="https://msdn.microsoft.com/37cfc0f8-8e17-4944-85fc-cc80ebff13df">Brush</a> object that contains an 
			<a href="https://msdn.microsoft.com/3732095d-c812-4ce5-80f1-9b191b4ff01c">Image</a> object that is used for the fill. The fill image can be transformed by using the local 
			<a href="https://msdn.microsoft.com/92b0d9db-3d4c-47b8-87cd-60d7b4323f0a">Matrix</a> object contained in the <b>Brush</b> object.
