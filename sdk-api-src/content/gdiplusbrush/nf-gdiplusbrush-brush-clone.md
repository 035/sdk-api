# Brush::Clone


## -description


The <b>Brush::Clone</b> method creates a new 
			<a href="https://msdn.microsoft.com/37cfc0f8-8e17-4944-85fc-cc80ebff13df">Brush</a> object based on this brush.


## -parameters






## -returns



Type: <strong>Type: <b><a href="https://msdn.microsoft.com/37cfc0f8-8e17-4944-85fc-cc80ebff13df">Brush</a>*</b>
</strong>

This method returns a pointer to the new 
						<a href="https://msdn.microsoft.com/37cfc0f8-8e17-4944-85fc-cc80ebff13df">Brush</a> object.
