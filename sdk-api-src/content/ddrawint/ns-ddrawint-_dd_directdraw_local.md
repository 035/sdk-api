# _DD_DIRECTDRAW_LOCAL structure


## -description


The DD_DIRECTDRAW_LOCAL structure contains driver information that is relevant to the current DirectDraw process only.


## -struct-fields




### -field lpGbl

Points to the <a href="https://msdn.microsoft.com/library/windows/hardware/ff550586">DD_DIRECTDRAW_GLOBAL</a> structure that describes the driver's device.
