# _DD_SURFACE_INT structure


## -description


The DD_SURFACE_INT structure contains the DirectDrawSurface object's interface information.


## -struct-fields




### -field lpLcl

Points to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff551733">DD_SURFACE_LOCAL</a> structure that describes a unique surface object.
