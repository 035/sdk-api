# __MIDL_IBackgroundCopyJob2_0001 enumeration


## -description


The 
<b>BG_AUTH_TARGET</b> enumeration defines the constant values that specify whether the credentials are used for proxy or server user authentication requests.


## -enum-fields




### -field BG_AUTH_TARGET_SERVER

Use credentials for server requests.


### -field BG_AUTH_TARGET_PROXY

Use credentials for proxy requests.
