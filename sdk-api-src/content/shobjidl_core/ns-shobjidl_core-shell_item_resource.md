# SHELL_ITEM_RESOURCE structure


## -description


Defines Shell item resource.


## -struct-fields




### -field guidType

Type: <b>GUID</b>

The <b>GUID</b> that identifies the item.


### -field szName

Type: <b>WCHAR[MAX_PATH]</b>

The item name. A null-terminated Unicode buffer of size MAX_LENGTH characters.
