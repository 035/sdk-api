# ISharingConfigurationManager::ArePrintersShared


## -description


Determines whether any printers connected to this computer are shared.


## -parameters






## -returns



Type: <b>HRESULT</b>

Returns standard HRESULT values, including the following:

</td>
</tr>

</dl>

</td>
</tr>
</table>
