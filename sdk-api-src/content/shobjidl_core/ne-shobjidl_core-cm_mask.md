# CM_MASK enumeration


## -description



          Indicates which values in the <a href="https://msdn.microsoft.com/b4437aa7-9682-4819-a353-936179e84005">CM_COLUMNINFO</a> structure should be set during calls to <a href="https://msdn.microsoft.com/3a52d634-0ff0-4dbc-81cb-90cdffe4f6ae">IColumnManager::SetColumnInfo</a>.


## -enum-fields




### -field CM_MASK_WIDTH

The <b>uWidth</b> member is specified.


### -field CM_MASK_DEFAULTWIDTH

The <b>uDefaultWidth</b> member is specified.


### -field CM_MASK_IDEALWIDTH

The <b>uIdealWidth</b> member is specified.


### -field CM_MASK_NAME

The <b>wszName</b> member is specified.


### -field CM_MASK_STATE

The <b>dwState</b> member is specified.
