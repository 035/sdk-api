# __MIDL___MIDL_itf_msopc_0001_0076_0002 enumeration


## -description


Describes how to interpret the  <i>selectionCriterion</i> parameter of the <a href="https://msdn.microsoft.com/ac1f0347-9b89-4d8f-b0cb-14708e7a6e55">IOpcRelationshipSelector::GetSelectionCriterion</a> method.


## -enum-fields




### -field OPC_RELATIONSHIP_SELECT_BY_ID

The <i>selectionCriterion</i> parameter is a relationship identifier.


### -field OPC_RELATIONSHIP_SELECT_BY_TYPE

The <i>selectionCriterion</i> parameter is a relationship type.
