# _MC_GAIN_TYPE enumeration


## -description



        Specifies whether to get or set a monitor's red, green, or blue gain.
      


## -enum-fields




### -field MC_RED_GAIN


            Red gain.
          


### -field MC_GREEN_GAIN


            Green gain.
          


### -field MC_BLUE_GAIN


            Blue gain.
