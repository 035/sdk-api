# ITextRow::SetProperty


## -description


Sets the value of the specified property. 


## -parameters




### -param Type [in]

Type: <b>long</b>

The ID of the  property to set.


### -param Value [in]

Type: <b>long</b>

The value.


## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">HRESULT</a></b>
