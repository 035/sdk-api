# ITextRow::GetCellColorFore


## -description


Gets the foreground color of the active cell.


## -parameters




### -param pValue [out, retval]

Type: <b>long*</b>

The foreground color.


## -returns



Type: <b><a href="https://msdn.microsoft.com/4553cafc-450e-4493-a4d4-cb6e2f274d46">HRESULT</a></b>
