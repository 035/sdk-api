# ITextRange2::SetPara2


## -description


Sets the paragraph format attributes of a range.


## -parameters




### -param pPara [in]

Type: <b><a href="https://msdn.microsoft.com/31a0849f-c651-4178-b1ff-a4333bcde5d9">ITextPara2</a>*</b>

The desired paragraph format.

Type: <b>HRESULT</b>
