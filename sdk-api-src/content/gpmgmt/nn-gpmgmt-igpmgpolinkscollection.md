# IGPMGPOLinksCollection interface


## -description


The 
<b>IGPMGPOLinksCollection</b> interface contains methods that enable applications to access a collection of GPO links when using the Group Policy Management (GPMC) interfaces.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IGPMGPOLinksCollection</b> interface inherits from the <a href="ebbff4bc-36b2-4861-9efa-ffa45e013eb5">IDispatch</a> interface. <b>IGPMGPOLinksCollection</b> also has these types of members:
<ul>
<li><a href="https://docs.microsoft.com/">Methods</a></li>
<li><a href="https://docs.microsoft.com/">Properties</a></li>
</ul>
