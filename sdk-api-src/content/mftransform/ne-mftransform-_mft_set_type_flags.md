# _MFT_SET_TYPE_FLAGS enumeration


## -description



Defines flags for the setting or testing the media type on a Media Foundation transform (MFT).




## -enum-fields




### -field MFT_SET_TYPE_TEST_ONLY

Test the proposed media type, but do not set it.
