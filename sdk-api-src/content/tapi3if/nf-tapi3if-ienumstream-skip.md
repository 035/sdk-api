# IEnumStream::Skip


## -description


The 
<b>Skip</b> method skips over the next specified number of elements in the enumeration sequence.


## -parameters




### -param celt [in]

Number of elements to skip.


## -returns



This method can return one of these values.

<dt><b>S_OK</b></dt>
</dl>
</td>

</tr>
