## -remarks



This enumeration is used by <a href="https://msdn.microsoft.com/eba25ade-e3a3-40fe-8555-2fcb07dc0ed3">ID3D10ShaderReflection1::GetGSInputPrimitive</a>.

This method requires Windows Vista Service Pack 1.

The   <b>D3D10_PRIMITIVE</b> enumeration is type defined in the  D3D10.h header file as a <a href="https://msdn.microsoft.com/d7a83edb-48ab-4e9f-bf2b-790ebb4a14c4">D3D_PRIMITIVE</a> enumeration, which is fully defined in the  D3DCommon.h header file.
