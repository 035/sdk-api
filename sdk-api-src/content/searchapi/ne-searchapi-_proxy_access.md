# _PROXY_ACCESS enumeration


## -description


Used by <a href="https://msdn.microsoft.com/23aeb0f6-857e-490c-9ede-11c0955a45c9">ISearchManager</a> to state proxy use.


## -enum-fields




### -field PROXY_ACCESS_PRECONFIG

Use proxy as set by Internet settings.


### -field PROXY_ACCESS_DIRECT

Do not use a proxy.


### -field PROXY_ACCESS_PROXY

Use the specified proxy.
