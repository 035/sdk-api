# __MIDL_IAnchor_0001 enumeration


## -description


Elements of the <b>TsGravity</b> enumeration specify the gravity type associated with an <a href="https://msdn.microsoft.com/a7d52959-8386-464f-958d-c870f286b265">IAnchor</a> object.


## -enum-fields




### -field TS_GR_BACKWARD

The anchor has backward gravity. For more information about anchor gravity, see <a href="ranges.htm">Ranges</a>.


### -field TS_GR_FORWARD

The anchor has forward gravity. For more information about anchor gravity, see <a href="ranges.htm">Ranges</a>.
