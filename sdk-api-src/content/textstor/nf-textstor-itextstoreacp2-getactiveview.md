# ITextStoreACP2::GetActiveView


## -description


Gets a <a href="https://msdn.microsoft.com/e649b799-d0d6-4f2d-b9f1-28070eea9b16">TsViewCookie</a> that represents the current active view.


## -parameters




### -param pvcView [out]

Receives the <b>TsViewCookie</b> data type that specifies the current active view.


## -returns



This method has no return values.
