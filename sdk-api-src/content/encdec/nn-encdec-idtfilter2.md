# IDTFilter2 interface


## -description



The <b>IDTFilter2</b> interface extends the <a href="https://msdn.microsoft.com/15acf764-7e4d-40c3-b907-ff5dfaa69dae">IDTFilter</a> interface and is exposed by the Decrypter/Detagger filter.
