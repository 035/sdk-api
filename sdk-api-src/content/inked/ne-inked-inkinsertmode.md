# InkInsertMode enumeration


## -description



Specifies how ink is inserted onto the <a href="https://msdn.microsoft.com/52761cb2-4433-4824-ba19-fe597de2faf0">InkEdit</a> control.




## -enum-fields




### -field IEM_InsertText


### -field IEM_InsertInk




#### - IEM_InsertAsInk

Drawn ink is inserted as ink.


#### - IEM_InsertAsText

Default. Drawn ink is inserted as text.
