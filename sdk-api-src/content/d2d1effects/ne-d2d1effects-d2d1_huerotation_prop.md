# D2D1_HUEROTATION_PROP enumeration


## -description


Identifiers for properties of the <a href="https://msdn.microsoft.com/D322DB2C-2B8B-4101-BFB2-97E49CAC7BF6">Hue rotate effect</a>.


## -enum-fields




### -field D2D1_HUEROTATION_PROP_ANGLE

The angle to rotate the hue, in degrees.
          

The type is FLOAT.

The default is 0.0f.
