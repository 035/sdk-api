# ISClusNetInterfaces::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Returns the 
    number of objects in the 
    <a href="https://msdn.microsoft.com/7d0dc4fd-733c-4a2a-9136-7dc0089b213d">ClusNetInterfaces</a> 
    collection.

This property is read-only.
