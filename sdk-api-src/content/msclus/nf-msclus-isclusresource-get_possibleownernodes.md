# ISClusResource::get_PossibleOwnerNodes


## -description


<p class="CCE_Message">[The 
    <b>PossibleOwnerNodes</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the <a href="https://msdn.microsoft.com/4381e378-7bf2-4dbc-b56e-3fed33193d32">nodes</a> specified as possible owners of a 
    <a href="https://msdn.microsoft.com/090d1c20-fab3-43dd-bfe2-a2c3f9ba8f89">resource</a>.

This property is read-only.
