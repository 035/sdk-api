# ISClusResDependencies::get_Item


## -description


<p class="CCE_Message">[The <b>Item</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Retrieves a 
    single <a href="https://msdn.microsoft.com/090d1c20-fab3-43dd-bfe2-a2c3f9ba8f89">resource</a> from a 
    <a href="https://msdn.microsoft.com/10695840-38ec-4614-8bbd-5772a53dea4b">ClusResDependencies</a> 
    collection.

This property is read-only.
