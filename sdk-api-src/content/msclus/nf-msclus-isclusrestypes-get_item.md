# ISClusResTypes::get_Item


## -description

    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

    <a href="https://msdn.microsoft.com/e4ad6364-f318-47f9-a276-d99c91ffbbb5">ClusResType</a> object from a
