# ISClusApplication::get_ClusterNames


## -description


<p class="CCE_Message">[The 
    <b>ClusterNames</b> property is available for 
    use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent 
    versions.]

Returns  the 
    names of all the <a href="c_gly.htm">clusters</a> in a 
    domain.

This property is read-only.
