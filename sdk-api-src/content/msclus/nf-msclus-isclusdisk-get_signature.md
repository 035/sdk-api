# ISClusDisk::get_Signature


## -description


<p class="CCE_Message">[The <b>Signature</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Returns the 
    signature of a <a href="https://msdn.microsoft.com/d42e9bca-3717-44f7-a1b9-dfad1dbddd23">Physical Disk</a>.

This property is read-only.
