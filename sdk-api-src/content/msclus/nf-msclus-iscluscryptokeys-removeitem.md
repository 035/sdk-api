# ISClusCryptoKeys::RemoveItem


## -description


<p class="CCE_Message">[The <b>RemoveItem</b> 
    method is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Deletes a 
    crypto key from a <a href="https://msdn.microsoft.com/0078ba7a-24d7-4de6-af05-f1a03d9deb0a">ClusCryptoKeys</a> 
    collection.


## -parameters




### -param varIndex

A <b>Variant</b> specifying the object to remove by name or by numeric index.


## -returns



This method does not return a value.
