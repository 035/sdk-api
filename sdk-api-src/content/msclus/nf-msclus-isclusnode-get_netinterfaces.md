# ISClusNode::get_NetInterfaces


## -description


<p class="CCE_Message">[The <b>NetInterfaces</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns 
    a <a href="https://msdn.microsoft.com/d40bd8bd-b822-4069-bc9c-b7fefc66c8d0">ClusNodeNetInterfaces</a> 
    collection providing access to the <a href="https://msdn.microsoft.com/cc0cbbc3-e342-483e-9c94-4ee43f4d588d">network interfaces</a> 
    installed on a <a href="https://msdn.microsoft.com/4381e378-7bf2-4dbc-b56e-3fed33193d32">node</a>.

This property is read-only.
