# ISClusResource::get_PrivateProperties


## -description


<p class="CCE_Message">[The 
    <b>PrivateProperties</b> property is available 
     for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
     subsequent versions.]

Returns the read/write <a href="https://msdn.microsoft.com/a1dee11c-f1fe-4509-a40a-a58c4b8999ef">private properties</a> of a 
    <a href="https://msdn.microsoft.com/090d1c20-fab3-43dd-bfe2-a2c3f9ba8f89">resource</a>.

This property is read-only.
