# ISDomainNames::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the number of objects 
    in the <a href="https://msdn.microsoft.com/e7f806a7-02e7-41be-b8b9-60351180e748">DomainNames</a> collection.

This property is read-only.
