# ISClusProperty::get_ValueCount


## -description


<p class="CCE_Message">[The <b>Format</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Returns the 
    number of <a href="https://msdn.microsoft.com/library/ms682858(v=VS.85).aspx">property values</a> associated with 
    a <a href="https://msdn.microsoft.com/library/ms682005(v=VS.85).aspx">cluster object</a> property.

This property is read-only.
