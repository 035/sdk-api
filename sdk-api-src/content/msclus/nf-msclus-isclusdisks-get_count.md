# ISClusDisks::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the number of 
    disks in the 
    <a href="https://msdn.microsoft.com/2823ccb2-5fb2-4e37-b842-340703165871">ClusDisks</a> collection.

This property is read-only.
