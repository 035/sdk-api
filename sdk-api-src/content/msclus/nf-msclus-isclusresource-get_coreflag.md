# ISClusResource::get_CoreFlag


## -description


<p class="CCE_Message">[The <b>CoreFlag</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns <a href="https://msdn.microsoft.com/54d00b1c-cef7-4310-8c10-743ee7086979">CLUS_FLAGS</a> flags associated with the 
    <a href="https://msdn.microsoft.com/090d1c20-fab3-43dd-bfe2-a2c3f9ba8f89">resource</a>.

This property is read-only.
