# ISClusResType::get_PossibleOwnerNodes


## -description


<p class="CCE_Message">[The 
    <b>PossibleOwnerNodes</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the <a href="https://msdn.microsoft.com/4381e378-7bf2-4dbc-b56e-3fed33193d32">nodes</a> specified as 
    <a href="/windows/desktop/dns/p-gly">possible owners</a> of a 
    <a href="https://msdn.microsoft.com/d02e4f51-7b86-451a-a51c-ea850ae464d1">resource type</a>.

This property is read-only.
