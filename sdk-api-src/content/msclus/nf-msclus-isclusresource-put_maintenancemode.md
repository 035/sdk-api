# ISClusResource::put_MaintenanceMode


## -description


<p class="CCE_Message">[The 
    <b>MaintenanceMode</b> property is available for 
    use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent 
    versions.]

Retrieves or sets the maintenance mode state of a disk resource.

This property is read/write.
