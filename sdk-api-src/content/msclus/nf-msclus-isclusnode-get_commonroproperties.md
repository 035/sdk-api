# ISClusNode::get_CommonROProperties


## -description


<p class="CCE_Message">[The 
    <b>CommonROProperties</b> property is available 
    for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

    the read-only <a href="https://msdn.microsoft.com/5341d390-69dd-4e84-a443-f35a4b6c0bab">common properties</a> of a
