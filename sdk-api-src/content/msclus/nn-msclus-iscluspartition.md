# ISClusPartition interface


## -description


<p class="CCE_Message">[The <b>ClusPartition</b> object 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Provides 
    information about a partition on a 
    <a href="https://msdn.microsoft.com/d42e9bca-3717-44f7-a1b9-dfad1dbddd23">Physical Disk</a> resource.


## -remarks



A <b>ClusPartition</b> object:

<ul>
<li>Is obtained from <a href="https://msdn.microsoft.com/62b58676-b026-428a-b035-9ad65e326a03">ClusPartitions.Item</a>.</li>
</ul>
