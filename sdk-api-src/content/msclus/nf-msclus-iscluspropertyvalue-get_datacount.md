# ISClusPropertyValue::get_DataCount


## -description


<p class="CCE_Message">[The DataCount property is available 
    for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the number of 
    <a href="/windows/desktop/dns/d-gly">data values</a> currently assigned to a 
    <a href="/windows/desktop/dns/p-gly">property value</a>.

This property is read-only.
