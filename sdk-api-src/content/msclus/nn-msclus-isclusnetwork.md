# ISClusNetwork interface


## -description


<p class="CCE_Message">[The <b>ClusNetwork</b> object is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Enables operations 
    on a <a href="https://msdn.microsoft.com/57d16e1f-e774-4ffb-b26b-7e72d6d589aa">network</a>, its properties, and related objects.


## -remarks



A <b>ClusNetwork</b> object:

<ul>
<li>Is obtained from 
      <a href="https://msdn.microsoft.com/7dba0985-16bc-4bda-9f37-275032c07811">ClusNetworks</a>.</li>
</ul>
