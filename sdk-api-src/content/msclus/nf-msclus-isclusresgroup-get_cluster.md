# ISClusResGroup::get_Cluster


## -description


<p class="CCE_Message">[The <b>Cluster</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Returns the 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn922625">cluster</a> associated with the 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn934674">group</a>.

This property is read-only.
