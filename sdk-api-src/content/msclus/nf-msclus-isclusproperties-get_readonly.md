# ISClusProperties::get_ReadOnly


## -description


<p class="CCE_Message">[The <b>ReadOnly</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Indicates 
    whether the properties in a <a href="https://msdn.microsoft.com/b117b0eb-e188-4514-8e11-9acca1303e8f">ClusProperties</a> 
    collection are <a href="https://msdn.microsoft.com/7df0981b-7253-40c1-9a5d-44cfc4cdf13b">read-only properties</a>.

This property is read-only.
