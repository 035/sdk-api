# ISClusDisk::get_Partitions


## -description


<p class="CCE_Message">[The <b>Partitions</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Returns a 
    <a href="https://msdn.microsoft.com/8a3c602a-a3b1-4b63-8dfa-36200c5c30ab">ClusPartitions</a> collection 
    containing partition information about a 
    <a href="https://msdn.microsoft.com/d42e9bca-3717-44f7-a1b9-dfad1dbddd23">Physical Disk</a>.

This property is read-only.
