# ISClusPartition::get_SerialNumber


## -description


<p class="CCE_Message">[The <b>SerialNumber</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns the 
    serial number of a 
    <a href="https://msdn.microsoft.com/library/ms682866(v=VS.85).aspx">storage class resource</a> 
    partition.

This property is read-only.
