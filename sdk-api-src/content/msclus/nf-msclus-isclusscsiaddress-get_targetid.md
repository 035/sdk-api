# ISClusScsiAddress::get_TargetId


## -description


<p class="CCE_Message">[The <b>TargetId</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns the 
    <a href="https://msdn.microsoft.com/library/ms682866(v=VS.85).aspx">SCSI</a> target ID of the 
    <a href="https://msdn.microsoft.com/d42e9bca-3717-44f7-a1b9-dfad1dbddd23">Physical Disk</a>.

This property is read-only.
