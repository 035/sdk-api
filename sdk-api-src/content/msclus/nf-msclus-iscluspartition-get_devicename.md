# ISClusPartition::get_DeviceName


## -description


<p class="CCE_Message">[The <b>DeviceName</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns the 
    device name of a 
    <a href="/windows/desktop/dns/s-gly">storage class resource</a> 
    partition.

This property is read-only.


## -parameters


## -remarks



The device name returned contains no backslashes.
