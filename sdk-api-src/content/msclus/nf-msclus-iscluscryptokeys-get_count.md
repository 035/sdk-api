# ISClusCryptoKeys::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the number 
    of crypto keys in the <a href="https://msdn.microsoft.com/0078ba7a-24d7-4de6-af05-f1a03d9deb0a">ClusCryptoKeys</a> 
    collection.

This property is read-only.
