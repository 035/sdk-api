# ISClusPartitions::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the number 
    of partitions in the <a href="https://msdn.microsoft.com/8a3c602a-a3b1-4b63-8dfa-36200c5c30ab">ClusPartitions</a> 
    collection.

This property is read-only.
