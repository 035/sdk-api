# ISClusPropertyValues::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property 
    is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
    in subsequent versions.]

Returns the 
    number of <a href="https://msdn.microsoft.com/library/ms682858(v=VS.85).aspx">property values</a> in the 
    <a href="https://msdn.microsoft.com/18ae71ee-5582-4ac9-bb0f-f1c077c0352a">ClusPropertyValues</a> 
    collection.

This property is read-only.
