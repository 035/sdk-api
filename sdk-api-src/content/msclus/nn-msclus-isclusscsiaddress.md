# ISClusScsiAddress interface


## -description


<p class="CCE_Message">[The <b>ClusScsiAddress</b> 
    object is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Provides access to 
    <a href="s_gly.htm">SCSI</a> information about a 
    <a href="https://msdn.microsoft.com/d42e9bca-3717-44f7-a1b9-dfad1dbddd23">Physical Disk</a>.


## -remarks



A <b>ClusScsiAddress</b> object is obtained from 
     <a href="https://msdn.microsoft.com/2f9bd5e9-a939-438d-9bd1-bd9ef30c53cd">ClusDisk.ScsiAddress</a>.
