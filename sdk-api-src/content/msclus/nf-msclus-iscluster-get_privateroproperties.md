# ISCluster::get_PrivateROProperties


## -description


<p class="CCE_Message">[The 
    <b>PrivateROProperties</b> property is available 
    for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns 
    the read-only <a href="https://msdn.microsoft.com/a1dee11c-f1fe-4509-a40a-a58c4b8999ef">private properties</a> of a 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn922625">cluster</a>.

This property is read-only.
