# ISClusNetwork::get_NetInterfaces


## -description


<p class="CCE_Message">[The <b>NetInterfaces</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns a 
    <a href="https://msdn.microsoft.com/8b99f752-8aee-4c43-b49a-8ebaf20220f2">ClusNetworkNetInterfaces</a> 
    collection providing access to the <a href="https://msdn.microsoft.com/cc0cbbc3-e342-483e-9c94-4ee43f4d588d">network interfaces</a> in 
    the <a href="https://msdn.microsoft.com/57d16e1f-e774-4ffb-b26b-7e72d6d589aa">network</a>.

This property is read-only.
