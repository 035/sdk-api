# ISClusNetworks::get_Item


## -description


<p class="CCE_Message">[The <b>Item</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Retrieves a single 
    <a href="https://msdn.microsoft.com/d51e9872-eb86-4b0e-b417-1a2e5ac6ee9c">ClusNetwork</a> object from the 
    <a href="https://msdn.microsoft.com/7dba0985-16bc-4bda-9f37-275032c07811">ClusNetworks</a> collection.

This property is read-only.
