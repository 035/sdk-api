# ISClusProperties::get_Count


## -description


<p class="CCE_Message">[The <b>Count</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the number 
    of properties in the <a href="https://msdn.microsoft.com/b117b0eb-e188-4514-8e11-9acca1303e8f">ClusProperties</a> 
    collection.

This property is read-only.
