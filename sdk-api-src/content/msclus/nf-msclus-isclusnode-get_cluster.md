# ISClusNode::get_Cluster


## -description


<p class="CCE_Message">[The <b>Cluster</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns a 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn922625">Cluster</a> object providing access to the 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn922625">cluster</a> associated with a 
    <a href="https://msdn.microsoft.com/4381e378-7bf2-4dbc-b56e-3fed33193d32">node</a>.

This property is read-only.
