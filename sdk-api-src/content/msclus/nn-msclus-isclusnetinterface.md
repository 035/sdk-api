# ISClusNetInterface interface


## -description


<p class="CCE_Message">[The <b>ClusNetInterface</b> 
    object is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Enables 
    operations on a <a href="https://msdn.microsoft.com/cc0cbbc3-e342-483e-9c94-4ee43f4d588d">network interface</a>, its properties, and 
    related objects.


## -remarks



A <b>ClusNetInterface</b> object is obtained from the following collections.

<ul>
<li>
<a href="https://msdn.microsoft.com/7d0dc4fd-733c-4a2a-9136-7dc0089b213d">ClusNetInterfaces</a>
</li>
<li>
<a href="https://msdn.microsoft.com/8b99f752-8aee-4c43-b49a-8ebaf20220f2">ClusNetworkNetInterfaces</a>
</li>
<li>
<a href="https://msdn.microsoft.com/d40bd8bd-b822-4069-bc9c-b7fefc66c8d0">ClusNodeNetInterfaces</a>
</li>
</ul>
