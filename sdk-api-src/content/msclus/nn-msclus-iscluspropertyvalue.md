# ISClusPropertyValue interface


## -description


<p class="CCE_Message">[The 
    <b>ClusPropertyValue</b> object is available for use 
    in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent 
    versions.]

Represents a single <a href="p_gly.htm">property value</a> 
    associated with a multi-value property. The property value consists of one or more 
    <a href="d_gly.htm">data values</a>.


## -remarks



A <b>ClusPropertyValue</b> object can only be 
    obtained from <a href="https://msdn.microsoft.com/8980e2ed-3822-4ecf-80d7-f3be42e92161">ClusPropertyValues.Item</a>.
