# ISClusNetInterface::get_Name


## -description


<p class="CCE_Message">[The <b>Name</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Retrieves or sets the 
    name of a <a href="https://msdn.microsoft.com/cc0cbbc3-e342-483e-9c94-4ee43f4d588d">network interface</a>.

This property is read-only.
