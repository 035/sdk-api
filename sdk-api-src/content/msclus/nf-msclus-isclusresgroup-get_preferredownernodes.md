# ISClusResGroup::get_PreferredOwnerNodes


## -description


<p class="CCE_Message">[The 
    <b>PreferredOwnerNodes</b> property is 
     available for use in the operating systems specified in the Requirements section. It may be altered or unavailable 
     in subsequent versions.]

Returns a 
    <a href="https://msdn.microsoft.com/3425825e-890c-4d3d-919e-a66963e1fc55">ClusResGroupPreferredOwnerNodes</a> 
    collection providing access to all <a href="https://msdn.microsoft.com/4381e378-7bf2-4dbc-b56e-3fed33193d32">nodes</a> on which a 
    <a href="https://msdn.microsoft.com/1e0680ba-87d0-4bf0-808c-d80485e4daa3">group</a> prefers to run.

This property is read-only.
