# ISClusPropertyValueData::get_Item


## -description


<p class="CCE_Message">[The <b>Item</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns a 
    single <a href="https://msdn.microsoft.com/en-us/library/Aa369367(v=VS.85).aspx">data value</a> from a 
    <a href="https://msdn.microsoft.com/d95a90f6-2a70-428b-aff3-3be9e9e66071">ClusPropertyValueData</a> 
    collection.

This property is read-only.
