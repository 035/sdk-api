# ISClusNetInterfaces::get_Item


## -description


<p class="CCE_Message">[The <b>Item</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Retrieves a 
    single <a href="https://msdn.microsoft.com/cc0cbbc3-e342-483e-9c94-4ee43f4d588d">network interface</a> from a 
    <a href="https://msdn.microsoft.com/7d0dc4fd-733c-4a2a-9136-7dc0089b213d">ClusNetInterfaces</a> 
    collection.

This property is read-only.
