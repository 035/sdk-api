# ISClusPropertyValueData::CreateItem


## -description


<p class="CCE_Message">[The 
    <b>CreateItem</b> method is available for use 
    in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent 
    versions.]

Creates a <a href="d_gly.htm">data value</a> and adds it to to a 
    <a href="https://msdn.microsoft.com/d95a90f6-2a70-428b-aff3-3be9e9e66071">ClusPropertyValueData</a> 
    collection.


## -parameters




### -param varValue

A <b>Variant</b> specifying the new data value.


### -param pvarData






## -returns



A <b>Variant</b> that receives the new data value.
