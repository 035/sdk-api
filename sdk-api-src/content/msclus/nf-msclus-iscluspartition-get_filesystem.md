# ISClusPartition::get_FileSystem


## -description


<p class="CCE_Message">[The <b>FileSystem</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns the 
    name of the file system residing on a 
    <a href="/windows/desktop/dns/s-gly">storage class resource</a> partition 
    ("FAT", "NTFS", and so on).

This property is read-only.
