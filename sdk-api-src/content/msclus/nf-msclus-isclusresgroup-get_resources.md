# ISClusResGroup::get_Resources


## -description


<p class="CCE_Message">[The <b>Resources</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Provides access to the 
    <a href="https://msdn.microsoft.com/090d1c20-fab3-43dd-bfe2-a2c3f9ba8f89">resources</a> in a 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn934674">group</a>.

This property is read-only.
