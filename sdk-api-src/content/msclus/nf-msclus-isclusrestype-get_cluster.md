# ISClusResType::get_Cluster


## -description


<p class="CCE_Message">[The <b>Cluster</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns the 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn922625">Cluster</a> object associated with the 
    <a href="https://msdn.microsoft.com/d02e4f51-7b86-451a-a51c-ea850ae464d1">resource type</a>.

This property is read-only.
