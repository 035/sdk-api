# ISClusApplication::get_DomainNames


## -description


<p class="CCE_Message">[The <b>DomainNames</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns the 
    names of all domains available to a 
    <a href="https://msdn.microsoft.com/2f837658-18f5-44c9-b743-1980a616e71a">ClusApplication</a> object.

This property is read-only.
