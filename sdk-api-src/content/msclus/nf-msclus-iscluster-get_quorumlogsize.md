# ISCluster::get_QuorumLogSize


## -description


<p class="CCE_Message">[The <b>QuorumLogSize</b> 
    property is available for use in the operating systems specified in the Requirements section. It may be altered or 
    unavailable in subsequent versions.]

Returns or sets the 
    maximum size of the log file maintained by the 
    <a href="https://msdn.microsoft.com/4c2ee30e-4de2-44ba-93ba-d2d89196545e">quorum resource</a>.

This property is read/write.
