# ISClusResGroup::get_CommonProperties


## -description


<p class="CCE_Message">[The 
    <b>CommonProperties</b> property is available 
    for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

    the read/write <a href="https://msdn.microsoft.com/5341d390-69dd-4e84-a443-f35a4b6c0bab">common properties</a> of a 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn934674">group</a>.

This property is read-only.
