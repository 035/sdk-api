# ISClusPropertyValue::get_Data


## -description


<p class="CCE_Message">[The <b>Data</b> property is available for use in the 
    operating systems specified in the Requirements section. It may be altered or unavailable in subsequent 
    versions.]

Retrieves a 
    <a href="https://msdn.microsoft.com/d95a90f6-2a70-428b-aff3-3be9e9e66071">ClusPropertyValueData</a> 
    collection containing all of the <a href="d_gly.htm">data values</a> associated 
    with a <a href="p_gly.htm">property value</a>.

This property is read-only.
