# ISClusterNames::get_Item


## -description


<p class="CCE_Message">[The <b>Item</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns a name of a 
    <a href="https://msdn.microsoft.com/library/windows/hardware/dn922625">cluster</a> from the 
    <a href="https://msdn.microsoft.com/c4e29498-c4e2-4351-8eed-05bc73437485">ClusterNames</a> collection.

This property is read-only.
