# ISClusNodes::get_Item


## -description


<p class="CCE_Message">[The <b>Item</b> property is 
    available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in 
    subsequent versions.]

Returns a single 
    <a href="https://msdn.microsoft.com/4381e378-7bf2-4dbc-b56e-3fed33193d32">node</a> from a 
    <a href="https://msdn.microsoft.com/f35d610f-014a-48cf-aaa4-93e320bcd890">ClusNodes</a> collection.

This property is read-only.
