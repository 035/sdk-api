# ID3D10EffectDepthStencilViewVariable::GetDepthStencil


## -description


Get a depth-stencil-view resource.


## -parameters




### -param ppResource [out]

Type: <b><a href="https://msdn.microsoft.com/library/Bb173526(v=VS.85).aspx">ID3D10DepthStencilView</a>**</b>

The address of a pointer to a depth-stencil-view interface. See <a href="https://msdn.microsoft.com/library/Bb173526(v=VS.85).aspx">ID3D10DepthStencilView Interface</a>.


## -returns



Type: <b><a href="https://msdn.microsoft.com/library/Hh437604(v=VS.85).aspx">HRESULT</a></b>

Returns one of the following <a href="https://msdn.microsoft.com/library/Bb205278(v=VS.85).aspx">Direct3D 10 Return Codes</a>.
