# ID3D10EffectConstantBuffer::SetTextureBuffer


## -description


Set a texture-buffer.


## -parameters




### -param pTextureBuffer [in]

Type: <b><a href="https://msdn.microsoft.com/303076f3-6057-4f7c-9aa8-a6dd72235ecc">ID3D10ShaderResourceView</a>*</b>

A pointer to a shader-resource-view interface for accessing a texture buffer.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

Returns one of the following <a href="https://msdn.microsoft.com/7b67d428-d000-4c3e-adc1-b5fc67a15a6a">Direct3D 10 Return Codes</a>.
