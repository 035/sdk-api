# IMFMediaTimeRange::GetLength


## -description


Gets the number of time ranges contained in the object.


## -parameters






## -returns



Returns the number of time ranges.




## -remarks



This method corresponds to the <b>TimeRanges.length</b> attribute in HTML5.
