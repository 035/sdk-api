# IMFMediaEngineEx::IsStereo3D


## -description


Queries whether the media resource contains stereoscopic 3D video.


## -parameters






## -returns



Returns <b>TRUE</b> if the media resource contains 3D video, or <b>FALSE</b> otherwise.
