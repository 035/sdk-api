# IMFTimedTextStyle::IsExternal method


## -description


Determines whether the timed-text style is external.


## -parameters






## -returns



Type: <b>BOOL</b>

Returns whether the timed-text style is external. If <b>TRUE</b>, the timed-text style is external; otherwise, <b>FALSE</b>.
