# IRecordInfo::RecordCreateCopy


## -description


Creates a copy of an instance of a record to the specified location.




## -parameters




### -param pvSource [in]

An instance of the record to be copied.



### -param ppvDest [out]

The new record with data copied from <i>pvSource</i>.
