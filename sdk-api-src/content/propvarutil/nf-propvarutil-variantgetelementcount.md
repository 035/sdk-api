# VariantGetElementCount function


## -description


Retrieves the element count of a variant structure.


## -parameters




### -param varIn [in]

Type: <b>REFVARIANT</b>

Reference to a source variant structure.


## -returns



Type: <b>ULONG</b>

Returns the element count for values of type VT_ARRAY; otherwise, returns 1.
