# IBasicVideo::put_SourceHeight method


## -description



The <code>put_SourceHeight</code> method sets the height of the source rectangle.




## -parameters




### -param SourceHeight [in]

Specifies the height, in pixels.


## -returns



Returns an <b>HRESULT</b> value.
