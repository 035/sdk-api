# ISWbemProperty interface


## -description


The 
<b>SWbemProperty</b> object represents a single WMI property of a managed object. This object cannot be created by the VBScript <a href="https://msdn.microsoft.com/3acbce31-6d56-4a7a-af03-fa37b2b868be">CreateObject</a> call.
