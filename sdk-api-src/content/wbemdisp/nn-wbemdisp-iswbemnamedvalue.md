# ISWbemNamedValue interface


## -description


The 
<b>SWbemNamedValue</b> object represents a single named value that belongs to the 
<a href="https://msdn.microsoft.com/7d1c3a28-d0d3-4108-9628-74ad483e328e">SWbemNamedValueSet</a> collection object. This object cannot be created by the VBScript <a href="https://msdn.microsoft.com/3acbce31-6d56-4a7a-af03-fa37b2b868be">CreateObject</a> call.
