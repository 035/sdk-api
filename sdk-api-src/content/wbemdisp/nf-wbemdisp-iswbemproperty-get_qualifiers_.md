# ISWbemProperty::get_Qualifiers_


## -description


The 
<b>Qualifiers_</b> property of the 
<a href="https://msdn.microsoft.com/2ddcfffa-a7b4-4fd6-926d-411de27f6212">SWbemProperty</a> object returns an 
<a href="https://msdn.microsoft.com/7ac5469c-357b-499d-a558-30bf760c5311">SWbemQualifierSet</a> object that is a collection of qualifiers for this WMI property. This property is read-only.

This property is read-only.
