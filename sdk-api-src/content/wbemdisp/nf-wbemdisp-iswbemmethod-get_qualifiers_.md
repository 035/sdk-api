# ISWbemMethod::get_Qualifiers_


## -description


The 
<b>Qualifiers_</b> property of the 
<a href="https://msdn.microsoft.com/461d5c41-4930-40cf-96e2-bc8cae335b60">SWbemMethod</a> object returns an 
<a href="https://msdn.microsoft.com/7ac5469c-357b-499d-a558-30bf760c5311">SWbemQualifierSet</a> object that is the collection of qualifiers for this method. This property is read-only.

This property is read-only.
