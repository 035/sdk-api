# ISWbemQualifier interface


## -description


You can use the properties of the 
<b>SWbemQualifier</b> object to represent a single qualifier of a WMI class, instance, property, or method parameter. This object cannot be created by the VBScript <a href="https://msdn.microsoft.com/3acbce31-6d56-4a7a-af03-fa37b2b868be">CreateObject</a> call.
