# FWP_DIRECTION_ enumeration


## -description


The FWP_DIRECTION enumeration type specifies the direction of network traffic.


## -enum-fields




### -field FWP_DIRECTION_OUTBOUND

Specifies outbound traffic.


### -field FWP_DIRECTION_INBOUND

Specifies inbound traffic.


### -field FWP_DIRECTION_MAX

The maximum value for this enumeration. This value might change in future versions of the NDIS
     header files and binaries.
