# FWP_BYTE_BLOB_ structure


## -description


The FWP_BYTE_BLOB structure defines a binary large object (BLOB).


## -struct-fields




### -field size

The number of bytes of binary data pointed to by the 
     <b>Data</b> member.


### -field data

A pointer to the binary data.


## -remarks



The FWP_BYTE_BLOB structure specifies a binary object of any size.
