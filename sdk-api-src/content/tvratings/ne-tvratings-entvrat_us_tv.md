# EnTvRat_US_TV enumeration


## -description



This topic applies to Windows XP Service Pack 1 or later.
        



The <b>EnTvRat_US_TV</b> enumeration type specifies rating levels in the US television rating system.


## -enum-fields




### -field US_TV_None

No rating level used.


### -field US_TV_Y

TVY — All children.


### -field US_TV_Y7

TVY7 — Directed to older children.


### -field US_TV_G

TVG — General audience.


### -field US_TV_PG

TVPG — Parental guidance suggested.


### -field US_TV_14

TV14 — Parents strongly cautioned.


### -field US_TV_MA

TVMA — Mature audience only.


### -field US_TV_None7

Not currently used.
