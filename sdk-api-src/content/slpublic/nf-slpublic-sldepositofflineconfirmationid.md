# SLDepositOfflineConfirmationId function


## -description


Deposits Installation ID (IID) and Confirmation ID (CID) for offline activation.


## -parameters




### -param hSLC [in]

Type: <b>HSLC</b>

The handle to the current SLC context.


### -param pProductSkuId [in]

Type: <b>const SLID*</b>

A pointer to the product ID.


### -param pwszInstallationId [in]

Type: <b>PCWSTR</b>

The Installation ID generated by the <a href="https://msdn.microsoft.com/2bfbedfc-6fac-468b-8314-c856aab856d0">SLGenerateOfflineInstallationId</a> function.


### -param pwszConfirmationId [in]

Type: <b>PCWSTR</b>

The confirmation ID CSR.


## -returns



Type: <b>HRESULT WINAPI</b>

If this function succeeds, it return <b>S_OK</b>.  Otherwise,  it returns an <b>HRESULT</b> error code.
