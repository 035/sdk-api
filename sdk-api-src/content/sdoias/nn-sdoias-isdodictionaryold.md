# ISdoDictionaryOld interface


## -description


Use the 
<b>ISdoDictionaryOld</b> interface to manipulate the dictionary of Remote Access Dial-In User Service (RADIUS) attributes.


## -inheritance

The <b xmlns:loc="http://microsoft.com/wdcml/l10n">ISdoDictionaryOld</b> interface inherits from the <a href="ebbff4bc-36b2-4861-9efa-ffa45e013eb5">IDispatch</a> interface. <b>ISdoDictionaryOld</b> also has these types of members:

</ul>
