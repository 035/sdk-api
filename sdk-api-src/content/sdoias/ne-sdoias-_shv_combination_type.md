# _SHV_COMBINATION_TYPE enumeration


## -description


The <b>SHV_COMBINATION_TYPE</b> enumeration type specifies the type of a System Health Validator (SHV) combination.


## -enum-fields




### -field SHV_COMBINATION_TYPE_ALL_PASS


### -field SHV_COMBINATION_TYPE_ALL_FAIL


### -field SHV_COMBINATION_TYPE_ONE_OR_MORE_PASS


### -field SHV_COMBINATION_TYPE_ONE_OR_MORE_FAIL


### -field SHV_COMBINATION_TYPE_ONE_OR_MORE_INFECTED


### -field SHV_COMBINATION_TYPE_ONE_OR_MORE_TRANSITIONAL


### -field SHV_COMBINATION_TYPE_ONE_OR_MORE_UNKNOWN


### -field SHV_COMBINATION_TYPE_MAX

Use this constant to test whether the value is in range.
