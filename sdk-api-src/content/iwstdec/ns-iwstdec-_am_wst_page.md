# _AM_WST_PAGE structure


## -description



The AM_WST_PAGE structure identifies a World Standard Teletext (WST) page.




## -struct-fields




### -field dwPageNr

The page number.


### -field dwSubPageNr

The sub-page number.


### -field pucPageData

A pointer to the page data.
