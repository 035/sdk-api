# Font::GetHeight


## -description


<span>This topic lists the 
			GetHeight  methods of the 
			<a href="https://msdn.microsoft.com/dd8af524-688c-44dd-b3e4-deadb874bdc3">Font</a> class. For a complete list of methods for the <b>Font</b> class, see <a href="https://msdn.microsoft.com/4b05c06c-1a67-4dcc-9ff7-b6c8280dc964">Font Methods</a>.

</span><h3>Overload list</h3><table>
<tr>
<th align="left" width="37%">Method</th>
<th align="left" width="63%">Description</th>
</tr>
<tr>
<td align="left" width="37%">
<a href="https://msdn.microsoft.com/7953300d-a2b9-4bef-977a-ca4e4d6fc49d">GetHeight(REAL)</a>
</td>
<td align="left" width="63%">
The <a href="https://msdn.microsoft.com/7953300d-a2b9-4bef-977a-ca4e4d6fc49d">Font::GetHeight</a> method gets the line spacing, in pixels, of this font. The line spacing is the vertical distance between the base lines of two consecutive lines of text. Thus, the line spacing includes the blank space between lines along with the height of the character itself.

</td>
</tr>
<tr>
<td align="left" width="37%">
<a href="https://msdn.microsoft.com/4cfe970e-332c-461b-9b75-0d0802fad86f">GetHeight(Graphics*)</a>
</td>
<td align="left" width="63%">
The <a href="https://msdn.microsoft.com/4cfe970e-332c-461b-9b75-0d0802fad86f">Font::GetHeight</a> method gets the line spacing of this font in the current unit of a specified <a href="https://msdn.microsoft.com/7e874710-3cd3-42c8-bd2f-8a779b19ba59">Graphics</a> object. The line spacing is the vertical distance between the base lines of two consecutive lines of text. Thus, the line spacing includes the blank space between lines along with the height of the character itself.

</td>
</tr>
</table>
