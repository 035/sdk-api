# Image::GetVerticalResolution


## -description


The <b>Image::GetVerticalResolution</b> method gets the vertical resolution, in dots per inch, of this image.


## -parameters






## -returns



Type: <strong>Type: <b>REAL</b>
</strong>

This method returns the vertical resolution, in dots per inch, of this image.
