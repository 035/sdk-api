# Image::GetFlags


## -description


The <b>Image::GetFlags</b> method gets a set of flags that indicate certain attributes of this 
			<a href="https://msdn.microsoft.com/3732095d-c812-4ce5-80f1-9b191b4ff01c">Image</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>UINT</b>
</strong>

This method returns a value that holds a set of single-bit flags. The flags are defined in the <a href="https://msdn.microsoft.com/52b1f348-0a4a-4e5a-b79c-2c75e247d238">ImageFlags</a> enumeration.
