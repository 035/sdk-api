# IFsrmSetting::get_MailFrom


## -description


Retrieves or sets the default email address from which email messages are sent.

This property is read/write.


## -parameters


## -remarks



The default is" FSRM@<i>local machine name</i>". You cannot set this to 
    "[Admin Email]".
