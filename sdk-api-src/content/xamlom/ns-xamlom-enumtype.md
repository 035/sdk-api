# EnumType structure


## -description


Represents a XAML Runtime enumeration.


## -struct-fields




### -field Name

The name of the enumeration.


### -field ValueInts

An array of int values in the enumeration.


### -field ValueStrings

An array of strings representing the named value of the enumeration.
