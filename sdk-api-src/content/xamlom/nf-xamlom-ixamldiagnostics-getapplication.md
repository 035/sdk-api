# IXamlDiagnostics::GetApplication


## -description


Gets an instance of the application.


## -parameters




### -param ppApplication [out, retval]

The application.


## -returns



If this method succeeds, it returns <b>S_OK</b>. Otherwise, it returns an <b>HRESULT</b> error code.
