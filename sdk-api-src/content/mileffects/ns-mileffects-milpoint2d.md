# MilPoint2D structure


## -description


Represents an x- and y-coordinate pair in two-dimensional space.


## -struct-fields




### -field X

Type: <b>DOUBLE</b>

The x-coordinate value of the point.


### -field Y

Type: <b>DOUBLE</b>

The y-coordinate value of the point.
