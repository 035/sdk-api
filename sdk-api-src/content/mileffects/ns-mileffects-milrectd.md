# MilRectD structure


## -description


Describes the width, height, and location of a rectangle.


## -struct-fields




### -field left

 


### -field top

 


### -field right

 


### -field bottom

 




#### - Height

Type: <b>DOUBLE</b>

The height of the rectangle.


#### - Width

Type: <b>DOUBLE</b>

The width of the rectangle.


#### - X

Type: <b>DOUBLE</b>

The location of the rectangle's left side.


#### - Y

Type: <b>DOUBLE</b>

The location of the rectangle's right side.
