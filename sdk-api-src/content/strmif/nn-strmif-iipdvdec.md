# IIPDVDec interface


## -description



The <code>IIPDVDec</code> interface provides methods for setting and retrieving properties on the <a href="https://msdn.microsoft.com/aa47010e-8510-475d-836a-cb63deeb3a7b">DV Video Decoder</a> filter.
