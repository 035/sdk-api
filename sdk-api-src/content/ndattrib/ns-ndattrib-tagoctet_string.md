# tagOCTET_STRING structure


## -description


The <b>OCTET_STRING</b> structure contains a pointer to a string of byte data.


## -struct-fields




### -field dwLength

Type: <b>DWORD</b>

The length of the data.


### -field lpValue

Type: <b>[size_is(dwLength)]BYTE*</b>

A pointer to the byte array containing the data.
