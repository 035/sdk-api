# ISCTE_EAS::GetDetailsOOBSourceID


## -description


The <b>GetDetailsOOBSourceID</b> method returns the source identifier of the virtual details channel for the emergency alert.


## -parameters




### -param pwVal [out]


            Receives the details_OOB_source_ID field.
          


## -returns
