# _WMUserText structure


## -description



The <b>WM_USER_TEXT</b> structure is used as the data item for the <a href="https://msdn.microsoft.com/892e7f02-6a21-49f2-a9cf-6ac52994a577">WM/Text</a> complex metadata attribute.




## -struct-fields




### -field pwszDescription

Pointer to a wide-character null-terminated string containing the description of the text entry.


### -field pwszText

Pointer to a wide-character null-terminated string containing the text.
