# RectF::GetBounds


## -description


The <b>RectF::GetBounds</b> method makes a copy of this rectangle.


## -parameters




### -param rect [out]

Type: <b>RectF*</b>

Pointer to a 
					<a href="https://msdn.microsoft.com/6821442b-d352-48cb-a48a-839105a8c36a">RectF</a> object that receives the copy of this rectangle. 


## -returns



This method does not return a value.
