# Rect::GetTop


## -description


The <b>Rect::GetTop</b> method gets the y-coordinate of the top edge of the rectangle.


## -parameters






## -returns



Type: <strong>Type: <b>INT</b>
</strong>

This method returns the y-coordinate of the top edge of the rectangle.
