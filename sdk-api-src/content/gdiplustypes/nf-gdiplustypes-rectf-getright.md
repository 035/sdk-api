# RectF::GetRight


## -description


The <b>RectF::GetRight</b> method gets the x-coordinate of the right edge of the rectangle.


## -parameters






## -returns



Type: <strong>Type: <b>REAL</b>
</strong>

This method returns the x-coordinate of the right edge of the rectangle.
