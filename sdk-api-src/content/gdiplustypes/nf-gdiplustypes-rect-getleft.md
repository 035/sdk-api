# Rect::GetLeft


## -description


The <b>Rect::GetLeft</b> method gets the x-coordinate of the left edge of the rectangle.


## -parameters






## -returns



Type: <strong>Type: <b>INT</b>
</strong>

This method returns the x-coordinate of the left edge of the rectangle.
