# IDXGIObject::GetParent


## -description


Gets the parent of the object.


## -parameters




### -param riid [in]

Type: <b>REFIID</b>

The ID of the requested interface.


### -param ppParent [out]

Type: <b>void**</b>

The address of a pointer to the parent object.


## -returns



Type: <b><a href="455d07e9-52c3-4efb-a9dc-2955cbfd38cc">HRESULT</a></b>

Returns one of the <a href="https://msdn.microsoft.com/9aa7dd65-6bf9-4731-8085-a9eab4224cdd">DXGI_ERROR</a> values.
