# IFELanguage::Open


## -description


Initializes the <a href="https://msdn.microsoft.com/9EE1BD9E-2D58-4720-841C-39865375BFE0">IFELanguage</a> object.

This method must be called before any use of the <a href="https://msdn.microsoft.com/9EE1BD9E-2D58-4720-841C-39865375BFE0">IFELanguage</a> object.


## -parameters






## -returns



<b>S_OK</b> if successful, otherwise <b>E_FAIL</b>.
