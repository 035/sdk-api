# IFECommon::InvokeWordRegDialog


## -description


Invokes the Microsoft IME Word Register Dialog Window from the app.


## -parameters




### -param pimedlg [in]

Pointer to an <a href="https://msdn.microsoft.com/14F39582-F51D-456F-BC19-AFE6E50D4155">IMEDLG</a> structure.


## -returns



<b>S_OK</b> if successful, 
otherwise <b>E_FAIL</b>.
