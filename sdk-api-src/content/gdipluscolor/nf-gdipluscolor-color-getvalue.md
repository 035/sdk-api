# Color::GetValue


## -description


The <b>Color::GetValue</b> method gets the <b>ARGB</b> value of this <a href="https://msdn.microsoft.com/library/windows/hardware/mt297756">Color</a> object.


## -parameters






## -returns



Type: <strong>Type: <b>ARGB</b>
</strong>

This method returns a 32-bit <b>ARGB</b> value that holds the alpha, red, green, and blue components of this color. <b>ARGB</b> is defined in Gdipluspixelformats.h.
