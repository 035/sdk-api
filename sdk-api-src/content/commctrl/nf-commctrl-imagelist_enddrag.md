# ImageList_EndDrag function


## -description


Ends a drag operation. 


## -parameters






## -returns



None. 




## -remarks



The temporary image list is destroyed when the <b>ImageList_EndDrag</b> function is called. To begin a drag operation, use the <a href="https://msdn.microsoft.com/5b262b32-5ec2-4f19-8a76-4c318aec7dc7">ImageList_BeginDrag</a> function.
