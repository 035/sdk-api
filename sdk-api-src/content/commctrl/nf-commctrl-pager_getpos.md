# Pager_GetPos macro


## -description


Retrieves the current scroll position of the pager control. You can use this macro or send the <a href="https://msdn.microsoft.com/1e0f967a-3290-43b7-b812-8cf56abf2d32">PGM_GETPOS</a> message explicitly.
