# X3DAUDIO_DISTANCE_CURVE_POINT structure


## -description


Defines a DSP setting at a given normalized distance.


## -struct-fields




### -field Distance

Normalized distance. This must be within 0.0f to 1.0f.


### -field DSPSetting

DSP control setting.
