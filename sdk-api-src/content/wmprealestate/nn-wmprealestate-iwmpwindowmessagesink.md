# IWMPWindowMessageSink interface


## -description




          This page documents a feature that may be unavailable in future versions of Windows Media Player and the Windows Media Player SDK.
        

The <b>IWMPWindowMessageSink</b> interface receives messages when the rendering plug-in does not have a window hosted in the Player UI.
