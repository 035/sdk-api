# IWMPNodeRealEstate::SetWindowless method


## -description




          This page documents a feature that may be unavailable in future versions of Windows Media Player and the Windows Media Player SDK.
        



The <b>SetWindowless</b> method receives a value indicating that the plug-in should render in windowless mode.


## -parameters




### -param fWindowless [in]

<b>Boolean</b>, true indicating windowless.


## -returns



The method returns an <b>HRESULT</b>.
