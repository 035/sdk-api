# ICertPropertyArchivedKeyHash::get_ArchivedKeyHash


## -description


The <b>ArchivedKeyHash</b> property retrieves a SHA-1 hash of the private key.

This property is read-only.


## -parameters


## -remarks



Call the <a href="https://msdn.microsoft.com/library/windows/hardware/ff550945">Initialize</a> method to specify the hash.
