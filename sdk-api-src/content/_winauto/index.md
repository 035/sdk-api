Overview of the Windows Automation API technology.

To develop Windows Automation API, you need these headers:

 * [uiautomationclient.h](..\uiautomationclient\index.md)
 * [uiautomationcore.h](..\uiautomationcore\index.md)
 * [uiautomationcoreapi.h](..\uiautomationcoreapi\index.md)

For the programming guide, see [Windows Automation API](https://review.docs.microsoft.com/en-us/win32-test/winauto).
