# IFolderBandPriv::SetAccelerators


## -description


Sets accelerators.


## -parameters




### -param fAccelerators

Type: <b>BOOL</b>

<b>TRUE</b> to enable accelerators; <b>FALSE</b> to disable them.


## -returns



Type: <b>HRESULT</b>

Always returns S_OK.
