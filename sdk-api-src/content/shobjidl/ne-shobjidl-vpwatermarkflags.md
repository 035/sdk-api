# VPWATERMARKFLAGS enumeration


## -description


Specifies watermark flags. Used by <a href="https://msdn.microsoft.com/14ce62f7-b464-4e52-8441-35f613b6c844">IVisualProperties::SetWatermark</a>.


## -enum-fields




### -field VPWF_DEFAULT

Default Windows XP behavior.


### -field VPWF_ALPHABLEND

Alpha blend the respective bitmap; assumed 24-bit color + 8-bit alpha.
